<div id="introduction">
  ## Introducción
</div>

La API Newsfeed de Benzinga ofrece noticias financieras en tiempo real y abarca una amplia variedad de tipos de contenido, incluida la función &quot;Why Is It Moving&quot; (WIIM). Admite filtros flexibles por:

* Tickers, ISIN y CUSIP
* Canales y temas
* Rangos de fechas y marcas de tiempo updated
* Tipos de contenido e importancia

<div id="available-endpoints">
  ## Endpoints disponibles
</div>

<div id="get-news">
  ### [Obtener noticias](/es/api-reference/news-api_api/news/get-the-news-items)
</div>

Devuelve datos estructurados de noticias.

<div id="get-news-channels">
  ### [Obtener canales de noticias](/es/api-reference/news-api_api/channels/get-available-news-channels)
</div>

Devuelve una lista de todos los canales de noticias disponibles para filtrar.

<div id="get-removed-news">
  ### [Obtener noticias eliminadas](/es/api-reference/news-api_api/news/returns-the-removed-news-data)
</div>

Devuelve noticias que han sido eliminadas o corregidas.

<div id="why-is-it-moving-wiim">
  ### [¿Por qué se está moviendo (WIIM)](/es/api-reference/news-api_api/news/get-the-news-items)
</div>

Acceda a las explicaciones características de Benzinga, de una sola frase, sobre movimientos relevantes de las acciones. Use el endpoint estándar de noticias para recuperar estos elementos.
