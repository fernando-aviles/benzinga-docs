<div id="introduction">
  ## Introducción
</div>

La API de Newsfeed de Benzinga proporciona noticias financieras en tiempo real e incluye una amplia variedad de tipos de contenido, entre ellos la función &quot;Why Is It Moving&quot; (WIIM). Admite filtros flexibles por:

* Tíkeres, ISIN y CUSIP
* Canales y temas
* Rangos de fechas y marcas de tiempo de última actualización
* Tipos de contenido e importancia

<div id="available-endpoints">
  ## Puntos finales disponibles
</div>

<div id="get-news">
  ### [Obtener noticias](/es/api-reference/news-api_api/news/get-the-news-items)
</div>

Devuelve datos estructurados de las noticias.

<div id="get-news-channels">
  ### [Obtener canales de noticias](/es/api-reference/news-api_api/channels/get-available-news-channels)
</div>

Devuelve una lista de todos los canales de noticias disponibles para filtrar.

<div id="get-removed-news">
  ### [Obtener noticias eliminadas](/es/api-reference/news-api_api/news/returns-the-removed-news-data)
</div>

Devuelve noticias que han sido eliminadas o corregidas.

<div id="why-is-it-moving-wiim">
  ### [Why Is It Moving (WIIM)](/es/api-reference/news-api_api/news/get-the-news-items)
</div>

Accede a las explicaciones características de Benzinga, de una sola frase, sobre movimientos relevantes de las acciones. Usa el punto final estándar de noticias para obtener estos elementos.
