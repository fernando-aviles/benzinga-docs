<div id="introduction">
  ## Introducción
</div>

La API de Ratios de ganancias ofrece ratios específicamente relacionados con las ganancias y la rentabilidad de la empresa, como:

* ROA (Rentabilidad sobre activos)
* ROE (Rentabilidad sobre el patrimonio)
* ROI (Retorno sobre la inversión)
