<div id="introduction">
  ## Introducción
</div>

La API de ratios de ganancia proporciona ratios específicamente relacionados con los resultados y la rentabilidad de la empresa, como:

* ROA (Rentabilidad sobre activos)
* ROE (Rentabilidad sobre el patrimonio)
* ROI (Retorno de la inversión)
