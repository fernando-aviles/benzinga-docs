<div id="introduction">
  ## Introducción
</div>

La API de Fundamentales de Empresas obtiene datos financieros completos necesarios para realizar un análisis profundo de una empresa. Incluye:

* **Estados financieros**: balance general, estado de resultados y estado de flujos de efectivo (original, reexpresado, TTM).
* **Perfil de la empresa**: descripción del negocio, sector, industria, ejecutivos y sede central.
* **Informes de resultados**: ingresos históricos, EPS, EBITDA y beneficio neto.
* **Clases de acciones y valoración**: detalles de las clases de acciones, alfa/beta y ratios de valoración.
