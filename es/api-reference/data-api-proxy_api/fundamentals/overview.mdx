<div id="introduction">
  ## Introducción
</div>

La API de Fundamentales de Empresas obtiene datos financieros completos necesarios para un análisis exhaustivo de empresas. Cubre:

* **Estados financieros**: balance general, estado de resultados, estado de flujo de efectivo (originales, reexpresados y TTM).
* **Perfil de la empresa**: descripción del negocio, sector, industria, ejecutivos y sede.
* **Informes de resultados**: ingresos históricos, EPS, EBITDA y beneficio neto.
* **Clase de acciones y valoración**: detalles de la clase de acciones, alfa/beta y ratios de valoración.
