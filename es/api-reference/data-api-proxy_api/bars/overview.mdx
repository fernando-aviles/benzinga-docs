<div id="introduction">
  ## Introducción
</div>

La API de Bars obtiene datos históricos de barras de precios OHLCV (Apertura, Máximo, Mínimo, Cierre y Volumen) para los valores especificados. Devuelve datos de precios agregados en función de los intervalos especificados y admite:

* Múltiples símbolos de tíker
* Varios rangos de tiempo (fechas absolutas y relativas)
* Agrupaciones temporales personalizables (1m, 5m, 1h, 1d, etc.)
* Filtrado por sesión de negociación (premercado, horario regular y after-hours)
