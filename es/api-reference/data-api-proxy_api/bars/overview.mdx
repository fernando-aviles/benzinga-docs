<div id="introduction">
  ## Introducción
</div>

La API de Bars obtiene datos históricos de barras de precios OHLCV (Apertura, Máximo, Mínimo, Cierre, Volumen) para los valores especificados. Devuelve datos de precios agregados según los intervalos especificados y admite:

* Múltiples tickers
* Varios rangos de tiempo (fechas absolutas y relativas)
* Bloques temporales personalizables (1m, 5m, 1h, 1d, etc.)
* Filtrado por sesión de negociación (pre-market, regular, after-hours)
