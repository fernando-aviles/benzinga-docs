<div id="introduction">
  ## Introducción
</div>

La API de Ratios de Valoración proporciona métricas clave para evaluar la valoración relativa de los valores. Entre ellas se incluyen ratios como:

* P/E (Precio-beneficio)
* P/S (Precio-ventas)
* P/B (Precio-valor contable)
