<div id="introduction">
  ## Introducción
</div>

La API de Insider Transactions proporciona datos sobre operaciones con acciones realizadas por insiders de la empresa, como directivos, miembros del consejo y principales accionistas. Incluye datos derivados de las presentaciones del Formulario 4 ante la SEC.

<div id="available-endpoints">
  ## Puntos finales disponibles
</div>

<div id="get-insider-transaction-filing">
  ### [Obtener presentación de operaciones de insiders](/es/api-reference/calendar_api/insider-transaction/insider-transaction-filing)
</div>

Devuelve presentaciones de operaciones de insiders, incluidas compras y ventas de acciones y ejercicios de opciones.

<div id="get-insider-owner">
  ### [Obtener titular insider](/es/api-reference/calendar_api/insider-transaction/insider-transaction-owner)
</div>

Devuelve información sobre los titulares insider.
