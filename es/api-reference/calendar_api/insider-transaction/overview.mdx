<div id="introduction">
  ## Introducción
</div>

La API de transacciones de insiders proporciona datos sobre operaciones bursátiles realizadas por personas con información privilegiada de la empresa, como directivos, consejeros y principales accionistas. Incluye datos derivados de los formularios 4 presentados ante la SEC.

<div id="available-endpoints">
  ## Endpoints disponibles
</div>

<div id="get-insider-transaction-filing">
  ### [Obtener declaración de transacciones de personas con información privilegiada](/es/api-reference/calendar_api/insider-transaction/insider-transaction-filing)
</div>

Devuelve declaraciones de transacciones de personas con información privilegiada, incluidas compras y ventas de acciones y ejercicios de opciones.

<div id="get-insider-owner">
  ### [Obtener propietario persona con información privilegiada](/es/api-reference/calendar_api/insider-transaction/insider-transaction-owner)
</div>

Devuelve información sobre los propietarios con información privilegiada.
