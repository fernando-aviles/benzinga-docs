<div id="introduction">
  ## Introducción
</div>

La API de transacción gubernamental rastrea la actividad bursátil de funcionarios del gobierno de EE. UU., incluidos miembros de la Cámara de Representantes y del Senado de EE. UU. Ofrece transparencia sobre:

* Informes periódicos de transacciones
* Registros de operaciones individuales
* Tipos de activos y montos de las transacciones

<div id="available-endpoints">
  ## Endpoints disponibles
</div>

<div id="get-government-trades">
  ### [Obtener transacción gubernamental](/es/api-reference/calendar_api/government-trades/government-trades)
</div>

Devuelve operaciones de funcionarios del gobierno, incluidas las transacciones de miembros del Congreso.

<div id="get-government-trade-reports">
  ### [Obtener informes sobre operaciones gubernamentales](/es/api-reference/calendar_api/government-trade-reports/government-trade-reports)
</div>

Devuelve informes detallados de divulgación sobre operaciones gubernamentales.
