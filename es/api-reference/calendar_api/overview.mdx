<div id="introduction">
  ## Introducción
</div>

El conjunto de API de Calendars proporciona acceso a una amplia variedad de datos de calendarios financieros, lo que ayuda a los inversores a seguir eventos corporativos y económicos clave.

<div id="available-groups">

| :--------------------------- | :-------------------------------------------------------------------- | :---------------------------------------------------------------------------------- |

| **Dividendos**               | Historial y calendario de pagos de dividendos                         | [Enlace](/es/api-reference/calendar_api/dividends/dividends-events-v22)                |
