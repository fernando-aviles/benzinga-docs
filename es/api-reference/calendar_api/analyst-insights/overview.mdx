<div id="introduction">
  ## Introducción
</div>

La API de Analyst Insights devuelve análisis sobre valores, incluidos análisis detallados y recomendaciones de analistas financieros. Proporciona acceso a:

* Mejoras y rebajas de calificación de analistas
* Precios objetivo
* Notas de investigación detalladas
* Contexto de las calificaciones de consenso
