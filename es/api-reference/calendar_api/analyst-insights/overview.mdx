<div id="introduction">
  ## Introducción
</div>

La API de Analyst Insights devuelve perspectivas de análisis sobre valores, incluidos análisis detallados y recomendaciones de analistas financieros. Proporciona acceso a:

* Mejoras y rebajas de recomendación de analistas
* Precios objetivo
* Notas de investigación detalladas
* Contexto de la calificación de consenso

  ## Puntos finales disponibles
</div>

<div id="get-analyst-insights">
  ### [Obtener Analyst Insights](/es/api-reference/calendar_api/analyst-insights/get-analyst-insights-v1)
</div>

Recupera perspectivas de analistas y enfoques de investigación para valores, firmas o analysts específicos.
