<div id="introduction">
  ## Introducción
</div>

La API de Desdoblamientos proporciona información sobre desdoblamientos de acciones históricos y próximos. Incluye detalles como:

* Relación de desdoblamiento
* Fecha de anuncio
* Fecha ex
* Fecha de pago

</div>

</div>

Devuelve los datos del calendario de desdoblamientos.
