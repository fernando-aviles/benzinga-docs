<div id="introduction">
  ## Introducción
</div>

La API Bulls Say Bears Say devuelve los casos de inversión alcistas y bajistas más recientes para un ticker determinado.

* **Casos alcistas**: Presentan argumentos positivos a favor de comprar una acción.
* **Casos bajistas**: Presentan argumentos negativos en contra de invertir en una acción.

Estos datos son útiles para comprender los puntos de vista opuestos sobre el posible rendimiento de un valor.
