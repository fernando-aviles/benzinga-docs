<div id="introduction">
  ## Introducción
</div>

La API Bulls Say Bears Say devuelve las tesis de inversión alcistas y bajistas más recientes para el tíker de una acción determinada.

* **Tesis alcistas**: Presentan argumentos positivos a favor de comprar una acción.
* **Tesis bajistas**: Presentan argumentos negativos en contra de invertir en una acción.

Estos datos son valiosos para comprender los puntos de vista opuestos sobre el posible rendimiento de un valor.
