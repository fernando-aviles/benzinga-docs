<div id="introduction">
  ## Introducción
</div>

La API de transcripciones de llamadas de resultados proporciona acceso a transcripciones completas y archivos de audio de llamadas de resultados corporativas. Incluye:

* Transcripciones completas de llamadas de resultados trimestrales y anuales
* Sesiones de preguntas y respuestas e identificación de los ponentes
* Marcas de tiempo para una navegación detallada
* URL descargables de archivos de audio
