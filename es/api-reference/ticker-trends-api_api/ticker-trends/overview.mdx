<div id="introduction">
  ## Introducción
</div>

La API de tendencias del ticker obtiene datos de tendencia para los tickers especificados en función de métricas de interacción de los usuarios. Ayuda a identificar valores que generan un mayor interés mediante el seguimiento de:

* Puntuaciones de tendencia
* Niveles de actividad
* Interacción en diferentes intervalos de tiempo
