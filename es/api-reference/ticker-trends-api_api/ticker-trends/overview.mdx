<div id="introduction">
  ## Introducción
</div>

La API de Tendencias de Tíkers obtiene datos de tendencias para los símbolos de tíker especificados a partir de las métricas de interacción de los usuarios. Ayuda a identificar los valores que generan un mayor interés mediante el seguimiento de:

* Puntuaciones de tendencia
* Niveles de actividad
* Interacción en distintos intervalos de tiempo
