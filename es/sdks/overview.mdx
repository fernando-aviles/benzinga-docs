Desde el desarrollo local hasta producción, los SDK de Benzinga ofrecen la forma más sencilla de que tu aplicación obtenga secretos de Benzinga cuando los necesite.

* Instala e inicializa en tu aplicación un SDK de cliente específico del lenguaje
* Configura para el cliente acceso con alcance limitado a un proyecto y entorno en Benzinga
* Obtén secretos cuando los necesites

<div id="faq">
  ## Preguntas frecuentes
</div>

<AccordionGroup>
  <Accordion title="¿Es eficiente hacer una solicitud de datos en cada llamada a la API?">
    El SDK de la API de Benzinga está optimizado para ofrecer un alto rendimiento. Almacena en caché los datos a los que se accede con frecuencia e implementa estrategias para reducir las solicitudes redundantes, lo que garantiza que su aplicación funcione de forma eficiente. Los detalles del almacenamiento en caché pueden variar según la implementación específica del SDK y el lenguaje de programación.
  </Accordion>

  <Accordion title="¿Qué sucede si falla una solicitud a la API de Benzinga?">
    El SDK incluye manejo de errores y mecanismos de reintento. Si una solicitud falla, el SDK realiza reintentos según la configuración o proporciona mensajes de error adecuados, lo que permite que su aplicación gestione los fallos correctamente.
  </Accordion>

  <Accordion title="¿Puedo personalizar la configuración del SDK para adaptarla a las necesidades de mi aplicación?">
    ¡Por supuesto! El SDK de la API de Benzinga permite la personalización mediante parámetros como la configuración de la clave de API, los tiempos de espera y otras opciones específicas de su caso de uso. Consulte la documentación del SDK para su lenguaje de programación para obtener más información sobre los parámetros configurables.
  </Accordion>
</AccordionGroup>
