Desde el desarrollo local hasta producción, los SDK de Benzinga ofrecen la forma más sencilla de hacer que tu aplicación obtenga secretos de Benzinga cuando los necesite.

* Instala e inicializa un SDK de cliente específico del lenguaje en tu aplicación
* Aprovisiona al cliente con acceso limitado a un proyecto y entorno en Benzinga
* Obtén secretos cuando sea necesario

<div id="faq">
  ## Preguntas frecuentes
</div>

<AccordionGroup>
  <Accordion title="¿Es eficiente solicitar datos en cada llamada a la API?">
    El SDK de la API de Benzinga está optimizado para ofrecer un buen rendimiento. Almacena en caché los datos a los que se accede con frecuencia e implementa estrategias para reducir las solicitudes redundantes, lo que garantiza que su aplicación funcione de forma eficiente. Los detalles del almacenamiento en caché pueden variar según la implementación específica del SDK y el lenguaje.
  </Accordion>

  <Accordion title="¿Qué sucede si falla una solicitud a la API de Benzinga?">
    El SDK incluye mecanismos de gestión de errores y reintento. Si una solicitud falla, el SDK realiza reintentos según la configuración o proporciona mensajes de error adecuados, lo que permite que su aplicación gestione los fallos correctamente.
  </Accordion>

  <Accordion title="¿Puedo personalizar la configuración del SDK para adaptarla a las necesidades de mi aplicación?">
    ¡Por supuesto! El SDK de la API de Benzinga permite la personalización mediante parámetros como la configuración de la clave de API, los tiempos de espera y otras opciones específicas de su caso de uso. Consulte la documentación del SDK para su lenguaje de programación para obtener más información sobre los parámetros configurables.
  </Accordion>
</AccordionGroup>
