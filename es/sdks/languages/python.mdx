<div id="overview">
  ## Descripción general
</div>

El SDK de Python de Benzinga proporciona acceso completo a las API de datos financieros y noticias de Benzinga para el análisis cuantitativo y la inteligencia de mercado. El SDK ofrece dos módulos principales, uno para datos financieros y otro para noticias, lo que permite a los desarrolladores crear aplicaciones financieras avanzadas.

<div id="installation">
  ## Instalación
</div>

Instala el SDK con pip usando SSH:

```bash
pip install git+ssh://git@github.com/Benzinga/benzinga-python-client.git
```

<div id="authentication">
  ## Autenticación
</div>

Todas las clases del SDK requieren una clave de API para autenticarse. Para obtener una clave de API, visite [cloud.benzinga.com](https://cloud.benzinga.com).

<div id="modules">
  ## Módulos
</div>

El SDK consta de dos módulos principales:

* **Financial Data** - Análisis cuantitativo financiero y datos de mercado
* **News Data** - Noticias financieras, titulares y sentimiento del mercado

***

<div id="financial-data-module">
  ## Módulo de datos financieros
</div>

<div id="initialization">
  ### Inicialización
</div>

```python
from benzingaorg import financial_data

fin = financial_data.Benzinga(api_key='YOUR_API_KEY')
```

<div id="price-market-data">
  ### Precios y datos de mercado
</div>

<div id="price-history">
  #### Historial de precios
</div>

Obtén velas diarias históricas para un intervalo de fechas especificado.

```python
data = fin.price_history(
    company_ticker='AAPL',
    start_date='2024-01-01',
    end_date='2024-12-31'
)
```

**Parámetros:**

* `company_ticker` (obligatorio) - Tíker bursátil
* `start_date` (obligatorio) - Fecha de inicio en formato YYYY-MM-DD
* `end_date` (obligatorio) - Fecha de fin en formato YYYY-MM-DD

<div id="quote">
  #### Cotización
</div>

Obtén datos de cotización en tiempo real, incluidos OHLC, bid/ask, volumen y porcentaje de variación.

```python
quote = fin.quote(company_ticker='AAPL')
```

**Devuelve:** símbolo, bolsa, type, nombre, apertura, máximo, mínimo, cierre, precio de compra, precio de venta, volumen, porcentaje de variación

<div id="chart">
  #### Gráfico
</div>

Obtén datos para gráficos con intervalos de tiempo personalizables.

```python
chart = fin.chart(
    company_ticker='AAPL',
    start_date='2024-01-01',
    end_date='2024-12-31',
    interval='1D'  # Opcional: 1MONTH, 1W, 1D, 1H, 15M
)
```

**Opciones de intervalo:** `1MONTH`, `1W`, `1D`, `1H`, `15M`

<div id="news-methods">

<div id="news">
  #### Noticias
</div>

Obtén artículos de noticias con opciones de filtrado y visualización flexibles.

```python
articles = news.news(
    pagesize=15,              # Predeterminado: 15
    display_output='full',    # Opciones: full, abstract, headline
    date_from='2024-01-01',   # Opcional
    date_to='2024-12-31',     # Opcional
    channel=None              # Opcional: Filtrar por canal
)
```

**Opciones de visualización de la salida:**

* `full` - Contenido completo del artículo
* `abstract` - Resumen del artículo
* `headline` - Solo titulares

<div id="output-formatting">
  ## Formato de salida
</div>

Ambos módulos admiten el método `.output()` para una mejor visualización del JSON:

```python
# Salida de datos financieros
data = fin.quote('AAPL')
fin.output(data)

# Salida de datos de noticias
articles = news.news(pagesize=10)
news.output(articles)
```

<div id="complete-example">
  ## Ejemplo completo
</div>

```python
from benzingaorg import financial_data, news_data

# Inicializar módulos
fin = financial_data.Benzinga(api_key='YOUR_API_KEY')
news = news_data.News(api_key='YOUR_API_KEY')

# Obtener datos financieros
quote = fin.quote('AAPL')
fundamentals = fin.fundamentals('AAPL')
earnings = fin.earnings(ticker='AAPL', date_from='2024-01-01')

# Obtener datos de noticias
articles = news.news(pagesize=20, display_output='full')
top_news = news.top_news(limit=5)

# Formatear y mostrar
fin.output(quote)
news.output(articles)
```
