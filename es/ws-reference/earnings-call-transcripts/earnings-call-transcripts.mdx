---
title: "Flujo de transcripciones de conferencias telefónicas"
description: "Este endpoint de WebSocket permite suscribirse a transcripciones en tiempo real de conferencias telefónicas."
---

<div id="websocket-websocketv1">
  ## WebSocket `websocket/v1`:
</div>

```txt
wss://api.benzinga.com/api/v1/earnings-call-transcripts/stream
```

<div id="parameters">
  ### Parámetros:
</div>

| Nombre    | Ubicación |  Tipo  | Obligatorio |                                                        Descripción |
| :-------- | :-------: | :----: | ----------: | -----------------------------------------------------------------: |
| `token`   |   query   | string |          Sí |                 token de WebSocket de Benzinga (bz.production***). |
| `tickers` |   query   | string |    Opcional | Lista de tickers a los que desea suscribirse, separados por comas. |

<div id="stream-response">
  ### Respuesta del flujo:
</div>

Al suscribirse al flujo, recibirá actualizaciones sobre las transcripciones de conferencias telefónicas en el siguiente formato:

```json
{
   "api_version":"EarningsCallTranscripts/v1",
   "kind":"stream_type",
   "data":{
      "type":"transcription_end",
      "content":{
         "call_id":"6757fbd6f0cc56000183c6a5",
         "transcript_chunk":"We are extremely pleased with our fourth quarter and full fiscal year 2024 results.",
         "security":{
            "company_name":"Limoneira",
            "ticker":"LMNR",
            "cik":"1342423",
            "isin":"US5327461043",
            "exchange":"NAS"
         },
         "time":1735189664,
         "index":0
      }
   }
}
```

<div id="responses">
  #### Respuestas
</div>

* **200 OK**
* **Descripción**: Éxito
* **Tipo de contenido**: `application/json`
* **Esquema**: [EarningsCallTranscriptsWSResp](#earningscalltranscriptswsresp)

<div id="earningscalltranscriptswsresp">
  ### EarningsCallTranscriptsWSResp
</div>

| Property        | Type   | Description                                                        |
| --------------- | ------ | ------------------------------------------------------------------ |
| api&#95;version | string | Versión de la API de WebSocket: `EarningsCallTranscripts/v1`.      |
| kind            | string | Tipo de flujo de la conexión WebSocket.                            |
| data            | object | Contiene detalles de los datos de WebSocket.                       |
| - type          | string | Tipo de mensaje (p. ej., `transcript_chunk`, `transcription_end`). |
| - content       | object | Referencia al esquema `EarningsCallTranscripts`.                   |

<div id="earningscalltranscripts">
  ### EarningsCallTranscripts
</div>

| Property             | Type    | Description                                                                              |
| -------------------- | ------- | ---------------------------------------------------------------------------------------- |
| call&#95;id          | string  | ID único de la llamada.                                                                  |
| transcript&#95;chunk | string  | Fragmento transcrito de la conferencia telefónica.                                       |
| security             | object  | Contiene los detalles del valor para el que se proporciona la información.               |
| - cik                | string  | Central Index Key del valor.                                                             |
| - exchange           | string  | Bolsa donde cotiza el valor.                                                             |
| - isin               | string  | Número Internacional de Identificación de Valores (ISIN).                                |
| - name               | string  | Nombre del valor.                                                                        |
| - symbol             | string  | Ticker del valor.                                                                        |
| time                 | integer | Marca temporal en la que se crea un fragmento de la transcripción (con formato float64). |
| index                | integer | Índice del fragmento de la transcripción (con formato int).                              |
