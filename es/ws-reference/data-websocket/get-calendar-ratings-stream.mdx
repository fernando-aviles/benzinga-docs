<div id="overview">
  ## Descripción general
</div>

Suscríbase a cambios en tiempo real en las calificaciones de analistas y a actualizaciones de precios objetivo. Supervise mejoras y rebajas de calificación, inicios de cobertura y ajustes de precios objetivo en cuanto se produzcan.

<div id="key-features">
  ### Características clave
</div>

* **Calificaciones en tiempo real**: Reciba notificaciones instantáneas sobre cambios en las calificaciones de analistas
* **Precios objetivo**: Realice un seguimiento de los precios objetivo actuales y anteriores, junto con sus ajustes
* **Detalles del analista**: Incluye el nombre del analista, la firma y métricas de precisión histórica
* **Clasificación de cambios**: Mejoras, rebajas, mantenimiento, inicio, reiteración
* **Filtrar por valor**: Suscríbase a tickers o ISIN específicos

<div id="use-cases">
  ### Casos de uso
</div>

* Crear alertas y notificaciones en tiempo real sobre cambios de calificación
* Hacer seguimiento del rendimiento y la precisión de las firmas de análisis
* Supervisar los cambios en el consenso de precios objetivo
* Crear dashboards de mejoras/rebajas de calificación
* Automatizar estrategias de trading basadas en acciones de analistas

<div id="quick-start">
  ## Inicio rápido
</div>

Usa la herramienta interactiva de prueba de WebSocket de arriba para conectarte y probar el stream en tiempo real.

<div id="connection-url">
  ### URL de conexión
</div>

```
wss://api.benzinga.com/api/v2.1/calendar/ratings/stream?token=YOUR_TOKEN
```

<div id="query-parameters">
  ### Parámetros de consulta
</div>

| Parámetro | Obligatorio | Descripción                                                 |
| --------- | ----------- | ----------------------------------------------------------- |
| `token`   | Sí          | Su token de la API WebSocket de Benzinga (bz.production***) |
| `tickers` | No          | Lista separada por comas de tickers (p. ej., `AAPL,TSLA`)   |
| `isins`   | No          | Lista separada por comas de ISIN para filtrar               |

<div id="rating-actions">
  ### Acciones de calificación
</div>

| Acción       | Descripción                                  |
| ------------ | -------------------------------------------- |
| `Upgrades`   | Calificación mejorada (p. ej., Neutral → Buy) |
| `Downgrades` | Calificación rebajada (p. ej., Buy → Hold)    |
| `Maintains`  | Calificación sin cambios                      |
| `Initiates`  | Inicio de cobertura                           |
| `Reiterates` | Confirma la calificación existente            |

<div id="message-actions">
  ### Acciones del mensaje
</div>

| Acción                | Descripción                  |
| --------------------- | ---------------------------- |
| `created` / `Created` | Nueva calificación publicada |
| `updated` / `Updated` | Calificación modificada      |
| `deleted` / `Deleted` | Calificación eliminada       |

<div id="interactive-commands">
  ## Comandos interactivos
</div>

| Comando  | Descripción                                                   |
| -------- | ------------------------------------------------------------- |
| `ping`   | Mantiene la conexión activa (responde con `pong`)             |
| `replay` | Reproduce hasta los últimos 100 mensajes almacenados en caché |

<div id="best-practices">
  ## Mejores prácticas
</div>

* **Seguimiento de cambios**: Compare `rating_current` vs `rating_prior` para detectar mejoras o rebajas de calificación
* **Movimiento del precio objetivo**: Supervise `pt_current` vs `pt_prior` para detectar cambios en el precio objetivo
* **Reputación del analista**: Considere la importancia de la firma y su precisión histórica
* **Filtro de importancia**: Céntrese en las calificaciones de alta importancia (4-5) para identificar las de mayor impacto
