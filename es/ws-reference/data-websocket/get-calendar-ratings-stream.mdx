<div id="overview">
  ## Descripción general
</div>

Suscríbete a los cambios en las calificaciones de analistas y a las actualizaciones de precios objetivo en tiempo real. Supervisa mejoras, rebajas, inicios de cobertura y ajustes de precios objetivo a medida que se producen.

<div id="key-features">
  ### Características clave
</div>

* **Calificaciones en tiempo real**: Recibe notificaciones instantáneas sobre cambios en las calificaciones de los analistas
* **Precios objetivo**: Sigue los precios objetivo actuales y anteriores, junto con sus ajustes
* **Detalles del analista**: Incluye el nombre del analista, la firma y métricas de precisión histórica
* **Clasificación de acciones**: Mejoras, rebajas, mantenimiento, inicio y reiteración
* **Filtrar por instrumento**: Suscríbete a tíkeres o ISIN específicos

<div id="use-cases">
  ### Casos de uso
</div>

* Crear alertas y notificaciones en tiempo real sobre cambios de calificación
* Hacer seguimiento del rendimiento y la precisión de las firmas de análisis
* Supervisar los cambios en el consenso del precio objetivo
* Crear paneles de subidas/bajadas de calificación
* Automatizar estrategias de trading basadas en los movimientos de los analistas

<div id="quick-start">
  ## Inicio rápido
</div>

Usa el probador interactivo de WebSocket de arriba para conectarte y probar el flujo en tiempo real.

<div id="connection-url">
  ### URL de conexión
</div>

```
wss://api.benzinga.com/api/v2.1/calendar/ratings/stream?token=YOUR_TOKEN
```

<div id="query-parameters">
  ### Parámetros de consulta
</div>

| Parámetro | Obligatorio | Descripción                                                 |
| --------- | ----------- | ----------------------------------------------------------- |
| `token`   | Sí          | Tu token de la API WebSocket de Benzinga (bz.production***) |
| `tickers` | No          | Lista de tíkeres separada por comas (p. ej., `AAPL,TSLA`)   |
| `isins`   | No          | Lista de ISIN separada por comas para filtrar               |

<div id="rating-actions">
  ### Acciones de calificación
</div>

| Acción       | Descripción                                   |
| ------------ | --------------------------------------------- |
| `Upgrades`   | Calificación mejorada (p. ej., Neutral → Buy) |
| `Downgrades` | Calificación rebajada (p. ej., Buy → Hold)    |
| `Maintains`  | Calificación sin cambios                      |
| `Initiates`  | Primera cobertura de calificación             |
| `Reiterates` | Confirma la calificación vigente              |

<div id="message-actions">
  ### Acciones del mensaje
</div>

| Acción                | Descripción                  |
| --------------------- | ---------------------------- |
| `created` / `Created` | Nueva calificación publicada |
| `updated` / `Updated` | Calificación modificada      |
| `deleted` / `Deleted` | Calificación eliminada       |

<div id="interactive-commands">
  ## Comandos interactivos
</div>

| Comando  | Descripción                                                   |
| -------- | ------------------------------------------------------------- |
| `ping`   | Mantiene la conexión activa (responde con `pong`)             |
| `replay` | Reproduce hasta los últimos 100 mensajes almacenados en caché |

<div id="best-practices">
  ## Mejores prácticas
</div>

* **Seguimiento de cambios**: Compare `rating_current` y `rating_prior` para detectar mejoras o rebajas de calificación
* **Movimiento del precio objetivo**: Supervise `pt_current` y `pt_prior` para identificar cambios en el precio objetivo
* **Reputación del analista**: Tenga en cuenta la importancia de la firma y su precisión histórica
* **Filtro de importancia**: Céntrese en las calificaciones de alta importancia (4-5) para identificar los cambios con mayor impacto
