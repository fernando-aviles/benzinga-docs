<div id="overview">
  ## Descripción general
</div>

Suscríbase a análisis en tiempo real de escenarios alcistas y bajistas para valores. Obtenga perspectivas equilibradas sobre escenarios tanto positivos (alcistas) como negativos (bajistas) para acciones.

<div id="key-features">
  ### Características clave
</div>

* **Análisis equilibrado**: Reciba casos alcistas y bajistas para obtener una visión integral del mercado
* **Actualizaciones en tiempo real**: Reciba notificaciones instantáneas cuando se creen, actualicen o eliminen casos
* **Filtrar por instrumento**: Suscríbase a tíkers o ISIN específicos
* **Sentimiento del mercado**: Comprenda tanto las perspectivas optimistas como las pesimistas

<div id="use-cases">
  ### Casos de uso
</div>

* Cree paneles de análisis de sentimiento que muestren ambas posturas
* Realice un seguimiento de los cambios en el sentimiento alcista/bajista a lo largo del tiempo
* Alerte a los usuarios sobre cambios significativos en la visión del mercado
* Proporcione a los clientes análisis de inversión equilibrados

<div id="quick-start">
  ## Inicio rápido
</div>

Usa el probador interactivo de WebSocket anterior para conectarte y probar el flujo en tiempo real.

<div id="connection-url">
  ### URL de conexión
</div>

```
wss://api.benzinga.com/api/v1/bulls_bears_say/stream?token=YOUR_TOKEN
```

<div id="query-parameters">
  ### Parámetros de consulta
</div>

| Parámetro | Obligatorio | Descripción                                                    |
| --------- | ----------- | -------------------------------------------------------------- |
| `token`   | Sí          | Tu token de la API de WebSocket de Benzinga (bz.production***) |
| `tickers` | No          | Lista de tíkers separados por comas (p. ej., `AAPL,TSLA`)      |
| `isins`   | No          | Lista de ISIN separados por comas para filtrar                 |

<div id="message-example">
  ## Ejemplo de mensaje
</div>

```json
{
  "id": "websocket123",
  "api_version": "websocket/v1",
  "kind": "bull_bear_case",
  "data": {
    "action": "created",
    "id": "case123",
    "timestamp": "2024-10-01T10:00:00Z",
    "content": {
      "bull_case": "Strong earnings growth and market expansion expected.",
      "bear_case": "Increased competition and regulatory pressures.",
      "ticker": "AAPL",
      "updated": 1696153200
    }
  }
}
```

<div id="message-actions">
  ### Acciones del mensaje
</div>

| Acción                | Descripción                          |
| --------------------- | ------------------------------------ |
| `created` / `Created` | Nuevo caso alcista/bajista publicado |
| `updated` / `Updated` | Caso existente modificado            |
| `deleted` / `Deleted` | Caso eliminado                       |

<div id="interactive-commands">
  ## Comandos interactivos
</div>

| Comando  | Descripción                                                 |
| -------- | ----------------------------------------------------------- |
| `ping`   | Mantiene la conexión activa (responde con `pong`)           |
| `replay` | Reenvía hasta los últimos 100 mensajes almacenados en caché |

<div id="best-practices">
  ## Mejores prácticas
</div>

* **Eliminación de duplicados**: Usa el campo `id` para evitar procesar mensajes duplicados
* **Heartbeat**: Envía comandos `ping` cada 30-60 segundos
* **Presentación equilibrada**: Muestra tanto los casos alcistas como los bajistas para ofrecer un análisis equilibrado
* **Seguimiento de actualizaciones**: Supervisa la marca de tiempo `updated` para identificar cambios recientes
