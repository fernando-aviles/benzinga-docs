<div id="overview">
  ## Descripción general
</div>

Suscríbase a análisis de analistas en tiempo real y reciba actualizaciones sobre calificaciones de analistas, precios objetivo y recomendaciones detalladas a medida que se publiquen.

<div id="key-features">
  ### Características clave
</div>

* **Actualizaciones en tiempo real**: Reciba notificaciones instantáneas cuando los analistas publiquen nuevos análisis o actualicen calificaciones existentes
* **Filtrar por instrumento**: Suscríbase a tíkers o ISIN específicos para recibir solo los datos relevantes
* **Datos completos**: Incluye detalles de la firma de análisis, calificaciones, precios objetivo y análisis detallados
* **Seguimiento de cambios**: Supervise cuándo se crean, actualizan o eliminan los análisis

<div id="use-cases">
  ### Casos de uso
</div>

* Sigue los cambios en el sentimiento de los analistas sobre los valores de tu cartera
* Crea alertas en tiempo real para mejoras o rebajas de recomendación
* Supervisa los cambios en el precio objetivo de varios analistas
* Agrega análisis de consenso de varias firmas de análisis

<div id="quick-start">
  ## Inicio rápido
</div>

Usa el tester interactivo de WebSocket de arriba para conectarte y probar el flujo en tiempo real.

<div id="connection-url">
  ### URL de conexión
</div>

```
wss://api.benzinga.com/api/v1/analyst/insights/stream?token=YOUR_TOKEN
```

<div id="query-parameters">
  ### Parámetros de consulta
</div>

| Parámetro | Obligatorio | Descripción                                                 |
| --------- | ----------- | ----------------------------------------------------------- |
| `token`   | Sí          | Tu token de la API WebSocket de Benzinga (bz.production***) |
| `tickers` | No          | Lista de tíkeres separada por comas (p. ej., `AAPL,MSFT`)   |
| `isins`   | No          | Lista de ISIN separada por comas para filtrar               |

<div id="message-actions">
  ### Acciones del mensaje
</div>

| Acción                | Descripción                         |
| --------------------- | ----------------------------------- |
| `created` / `Created` | Nuevo insight de analista publicado |
| `updated` / `Updated` | Insight existente modificado        |
| `deleted` / `Deleted` | Insight eliminado                   |

<div id="interactive-commands">
  ## Comandos interactivos
</div>

| Comando  | Descripción                                       |
| -------- | ------------------------------------------------- |
| `ping`   | Mantiene la conexión activa (responde con `pong`) |
| `replay` | Reenvía hasta los últimos 100 mensajes en caché   |

<div id="best-practices">
  ## Mejores prácticas
</div>

* **Eliminación de duplicados**: Usa el campo `id` para evitar procesar mensajes duplicados
* **Heartbeat**: Envía comandos `ping` cada 30-60 segundos para mantener la conexión
* **Manejo de errores**: Implementa una lógica de reconexión con retroceso exponencial
* **Filtrado**: Usa filtros de tíker/ISIN para reducir el ancho de banda
