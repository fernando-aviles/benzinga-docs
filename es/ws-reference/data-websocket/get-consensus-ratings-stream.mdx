<div id="overview">
  ## Descripción general
</div>

Suscríbase a las calificaciones de consenso en tiempo real, que consolidan múltiples opiniones de analistas en calificaciones y precios objetivo unificados. Siga el sentimiento general del mercado y los cambios en el consenso de los analistas.

<div id="key-features">
  ### Funcionalidades clave
</div>

* **Consenso agregado**: Vista combinada de todas las calificaciones de analistas para un valor
* **Consenso de precio objetivo**: Precio objetivo promedio, máximo y mínimo
* **Distribución de calificaciones**: Desglose por Compra fuerte, Compra, Mantener, Venta y Venta fuerte
* **Número de analistas**: Seguimiento de la cobertura total y de analistas únicos
* **Agregación flexible**: Visualización en recuentos o porcentajes

<div id="use-cases">
  ### Casos de uso
</div>

* Crear dashboards y widgets de calificaciones de consenso
* Hacer seguimiento de la evolución general del sentimiento de los analistas
* Comparar calificaciones individuales con el consenso
* Supervisar los rangos de precios objetivo (máximo, mínimo, promedio)
* Identificar valores cuyo consenso mejora o empeora

<div id="quick-start">
  ## Inicio rápido
</div>

Utiliza el probador interactivo de WebSocket de arriba para conectarte y probar el flujo en tiempo real.

<div id="connection-url">
  ### URL de conexión
</div>

```
wss://api.benzinga.com/api/v1/consensus-ratings/stream?token=YOUR_TOKEN
```

<div id="query-parameters">
  ### Parámetros de consulta
</div>

| Parámetro        | Obligatorio | Descripción                                                      |
| ---------------- | ----------- | ---------------------------------------------------------------- |
| `token`          | Sí          | Su token de la API WebSocket de Benzinga (bz.production***)      |
| `tickers`        | No          | Lista de ticker symbols separada por comas (p. ej., `AAPL,MSFT`) |
| `isins`          | No          | Lista de ISIN separados por comas para filtrar                   |
| `simplify`       | No          | Reduce las calificaciones a Buy, Hold y Sell                     |
| `aggregate_type` | No          | Tipo de agregación: `number` o `percentage`                      |

<div id="consensus-ratings">
  ### Calificaciones de consenso
</div>

| Calificación  | Valor numérico | Descripción          |
| ------------- | -------------- | -------------------- |
| `STRONG_BUY`  | 5.0            | Consenso muy alcista |
| `BUY`         | 4.0 - 4.9      | Consenso alcista     |
| `HOLD`        | 3.0 - 3.9      | Consenso neutro      |
| `SELL`        | 2.0 - 2.9      | Consenso bajista     |
| `STRONG_SELL` | 1.0 - 1.9      | Consenso muy bajista |

<div id="key-metrics">
  ### Métricas clave
</div>

| Campo                    | Descripción                                 |
| ------------------------ | ------------------------------------------- |
| `consensus_rating`       | Calificación general (BUY, HOLD, SELL)      |
| `consensus_rating_val`   | Valor numérico de la calificación (1.0-5.0) |
| `consensus_price_target` | Precio objetivo promedio de los analistas   |
| `high_price_target`      | Precio objetivo más alto                    |
| `low_price_target`       | Precio objetivo más bajo                    |
| `total_analyst_count`    | Número total de calificaciones              |
| `unique_analyst_count`   | Número de analistas distintos               |

<div id="interactive-commands">
  ## Comandos interactivos
</div>

| Comando  | Descripción                                       |
| -------- | ------------------------------------------------- |
| `ping`   | Mantiene la conexión activa (responde con `pong`) |
| `replay` | Reproduce hasta los últimos 100 mensajes en caché |

<div id="best-practices">
  ## Prácticas recomendadas
</div>

* **Cambios en el consenso**: Supervise los cambios en `consensus_rating_val` con el paso del tiempo
* **Rango de precio objetivo**: Compare el precio actual con los objetivos máximo y mínimo para contextualizar la valoración
* **Profundidad de cobertura**: Un mayor número de analistas indica un consenso más fiable
* **Análisis de distribución**: Examine el desglose de `aggregate_ratings` para captar mejor los matices del sentimiento
* **Vista porcentual**: Use `aggregate_type=percentage` para realizar comparaciones normalizadas
