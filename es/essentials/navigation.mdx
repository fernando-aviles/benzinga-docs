El menú de navegación es la lista de enlaces de un sitio web.

Probablemente tendrás que actualizar `docs.json` cada vez que agregues una página nueva. Las páginas no aparecen automáticamente.

<div id="navigation-syntax">
  ## Sintaxis de navegación
</div>

Nuestra sintaxis de navegación es recursiva, lo que significa que puedes crear grupos de navegación anidados. No necesitas incluir `.mdx` en los nombres de las páginas. Las pestañas agrupan grupos y los grupos agrupan páginas.

<CodeGroup>
  ```json Navegación estándar
  "navigation": {
    "tabs": [
      {
        "tab": "Documentation",
        "groups": [
          {
            "group": "Getting Started",
            "pages": ["quickstart"]
          }
        ]
      }
    ]
  }
  ```

  ```json Navegación anidada
  "navigation": {
    "tabs": [
      {
        "tab": "Documentation",
        "groups": [
          {
            "group": "Getting Started",
            "pages": [
              "quickstart",
              {
                "group": "Nested Reference Pages",
                "pages": ["nested-reference-page"]
              }
            ]
          }
        ]
      }
    ]
  }
  ```
</CodeGroup>

<div id="folders">
  ## Carpetas
</div>

Simplemente coloca tus archivos MDX en carpetas y actualiza las rutas en `docs.json`.

Por ejemplo, para tener una página en `https://yoursite.com/your-folder/your-page`, crea una carpeta llamada `your-folder` que contenga un archivo MDX llamado `your-page.mdx`.

<Warning>
  No puedes usar `api` como nombre de carpeta a menos que la anides dentro de otra carpeta. Mintlify usa Next.js, que reserva la carpeta `api` del nivel superior para llamadas internas del servidor. Se aceptaría un nombre de carpeta como `api-reference`.
</Warning>

```json Navegación With carpeta
"navigation": {
  "tabs": [
    {
      "tab": "Documentation",
      "groups": [
        {
          "group": "Group Name",
          "pages": ["your-folder/your-page"]
        }
      ]
    }
  ]
}
```

<div id="hidden-pages">
  ## Páginas ocultas
</div>

Los archivos MDX que no estén incluidos en `docs.json` no aparecerán en la barra lateral, pero se podrá acceder a ellos mediante la barra de búsqueda y enlaces directos.
