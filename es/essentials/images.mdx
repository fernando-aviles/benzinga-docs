  ## Imagen
</div>

<div id="using-markdown">
  ### Uso de Markdown
</div>

La [sintaxis de Markdown](https://www.markdownguide.org/basic-syntax/#images) permite agregar imágenes con el siguiente código

```md
![title](/path/image.jpg)
```

Ten en cuenta que el tamaño del archivo de imagen debe ser inferior a 5 MB. De lo contrario, recomendamos alojar la imagen en un servicio como [Cloudinary](https://cloudinary.com/) o [S3](https://aws.amazon.com/s3/). Después, puedes usar esa URL para incrustarla.

<div id="using-embeds">
  ### Uso de embeds
</div>

Para tener más opciones de personalización de las imágenes, también puedes usar [embeds](/es/writing-content/embed) para añadirlas

```html
<img height="200" src="/path/image.jpg" />
```

<div id="iframes">
  ### iFrames
</div>

Carga una página HTML dentro del documento. Se utiliza sobre todo para incrustar videos.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
