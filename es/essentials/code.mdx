<div id="inline-code">
  ### Código en línea
</div>

Para indicar una `word` o `phrase` como código, enciérrala entre acentos graves (`).

```
Para indicar que una `palabra` o `frase` es código, enciérrala entre acentos graves (`).
```

<div id="code-block">
  ### Bloque de código
</div>

Usa [bloques de código delimitados](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks): encierra el código entre tres acentos graves e indica a continuación el lenguaje de programación del fragmento para obtener resaltado de sintaxis. De forma opcional, también puedes escribir el nombre del código después del lenguaje de programación.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
