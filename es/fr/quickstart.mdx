<div id="setup-your-development">
  ## Configura tu entorno de desarrollo
</div>

Aprende a actualizar tu documentación de forma local y a publicarla.

<div id="edit-and-preview">
  ### Editar y previsualizar
</div>

<AccordionGroup>
  <Accordion icon="github" title="Clona tu documentación localmente">
    Durante el proceso de incorporación, creamos un repositorio en tu cuenta de GitHub con
    el contenido de tu documentación. Puedes encontrar este repositorio en nuestro
    [dashboard](https://dashboard.mintlify.com). Para clonar el repositorio
    localmente, sigue estas
    [instrucciones](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)
    en tu terminal.
  </Accordion>

  <Accordion icon="rectangle-terminal" title="Previsualiza los cambios">
    La vista previa te ayuda a comprobar que tus cambios se vean como esperas. Creamos una
    herramienta de línea de comandos para ver estos cambios localmente. 1. Instala la
    [Mintlify CLI](https://www.npmjs.com/package/mintlify) para previsualizar los
    cambios de la documentación localmente con este comando: `npm i -g mintlify`
    2. Ejecuta el siguiente comando en la raíz de tu documentación (donde
    está `docs.json`): `mintlify dev`
  </Accordion>
</AccordionGroup>

<div id="deploy-your-changes">
  ### Despliega tus cambios
</div>

<AccordionGroup>
  <Accordion icon="message-bot" title="Instala nuestra app de GitHub">
    Nuestra app de GitHub despliega automáticamente tus cambios en tu sitio de
    documentación, así que no necesitas gestionar los despliegues por tu cuenta.
    Puedes encontrar el enlace de instalación en tu [dashboard](https://dashboard.mintlify.com). Una vez que el bot se haya
    instalado correctamente, debería aparecer una marca de verificación junto al hash
    del commit del repositorio.
  </Accordion>

  <Accordion icon="rocket" title="Haz push de tus cambios">
    [Haz commit y push de tus cambios a
    Git](https://docs.github.com/en/get-started/using-git/pushing-commits-to-a-remote-repository#about-git-push)
    para que se actualicen en tu sitio de documentación. Si haces push y no ves que
    la app de GitHub haya desplegado correctamente tus cambios, también puedes
    actualizar manualmente tu documentación desde nuestro [dashboard](https://dashboard.mintlify.com).
  </Accordion>
</AccordionGroup>
