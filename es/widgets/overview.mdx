<div id="what-are-benzinga-widgets">
  ### ¿Qué son Benzinga Widgets?
</div>

Benzinga Widgets son componentes prediseñados e incrustables que te permiten mostrar gráficos financieros, información bursátil y otros datos de mercado en tu sitio web con una configuración mínima. Son totalmente personalizables y están diseñados para integrarse sin problemas con tu contenido existente.

<div id="features-of-benzinga-widgets">
  ### Características de Benzinga Widgets
</div>

<div id="interactive-stock-charts">
  #### 📈 Gráficos interactivos de acciones
</div>

* **Datos en tiempo real**: Muestra los precios de las acciones al instante.
* **Intervalos de tiempo personalizables**: Permite a tus usuarios explorar los datos en distintos períodos, como 1 día, 6 meses o 5 años.
* **Visualizaciones dinámicas**: Nuestros gráficos se actualizan automáticamente para reflejar las últimas tendencias del mercado.

```html
<iframe frameBorder="0" height="500" width="500" src="https://widgets.benzinga.com/embed/chart/AAPL?timeframe=6m&exchange=NASDAQ&bgColor=%23ffffff&chartColor=%232962FF" title="Apple Stock Chart" ></iframe>
```

<div id="how-to-use-benzinga-widgets-on-your-website">
  ### Cómo usar Benzinga Widgets en su sitio web
</div>

Siga estos sencillos pasos para integrar un widget de Benzinga:

1. **Elija su widget**
   Decida qué widget se adapta mejor a sus necesidades. Entre las opciones más populares se incluyen gráficos bursátiles, feeds de noticias y tíkeres en tiempo real.

2. **Personalice su código de inserción**
   Modifique la URL de inserción proporcionada para adaptarla a sus necesidades, por ejemplo, cambiando el símbolo de la acción, el período de tiempo o la combinación de colores. Por ejemplo:

```html
<iframe
    frameBorder="0"
    height="600"
    class="w-full"
    src="https://widgets.benzinga.com/embed/chart/AAPL?timeframe=6m&exchange=NASDAQ&bgColor=%23ffffff&chartColor=%232962FF"
></iframe>
```

3. **Pega el código en tu sitio web**
   Inserta el código del iframe en la sección deseada de tu HTML o CMS (p. ej., WordPress, Wix).

4. **Prueba la integración**
   Visita tu sitio web para comprobar que el widget se carga y se muestra correctamente. Ajusta las dimensiones y los estilos si es necesario.
