<div id="what-are-benzinga-widgets">
  ### ¿Qué son los widgets de Benzinga?
</div>

Los widgets de Benzinga son componentes prediseñados e integrables que te permiten mostrar gráficos financieros, información bursátil y otros datos de mercado en tu sitio web con una configuración mínima. Son totalmente personalizables y están diseñados para integrarse sin problemas con tu contenido existente.

<div id="features-of-benzinga-widgets">
  ### Características de los widgets de Benzinga
</div>

<div id="interactive-stock-charts">
  #### 📈 Gráficos interactivos de acciones
</div>

* **Datos en tiempo real**: Muestra precios de acciones en tiempo real.
* **Períodos personalizables**: Permite que tus usuarios exploren los datos en distintos períodos, como 1 día, 6 meses o 5 años.
* **Visualizaciones dinámicas**: Nuestros gráficos se actualizan automáticamente para reflejar las últimas tendencias del mercado.

```html
<iframe frameBorder="0" height="500" width="500" src="https://widgets.benzinga.com/embed/chart/AAPL?timeframe=6m&exchange=NASDAQ&bgColor=%23ffffff&chartColor=%232962FF" title="Apple Stock Chart" ></iframe>
```

<div id="additional-widgets">

* **Tickers bursátiles**: Muestra cotizaciones en tiempo real en un formato de ticker desplazable.
* **Feeds de Noticias**: Mantén a tu audiencia informada con las últimas noticias del mercado.
* **Información sobre empresas**: Proporciona estadísticas y datos clave sobre empresas específicas.

<div id="how-to-use-benzinga-widgets-on-your-website">
  ### Cómo usar los widgets de Benzinga en tu sitio web
</div>

Sigue estos sencillos pasos para integrar un widget de Benzinga:

1. **Elige tu widget**
   Decide qué widget se adapta mejor a tus necesidades. Entre las opciones más populares se incluyen gráficos bursátiles, feeds de noticias y tickers en tiempo real.

2. **Personaliza tu código de inserción**
   Modifica la URL de inserción proporcionada para adaptarla a tus necesidades, por ejemplo, cambiando el ticker de la acción, el intervalo de tiempo o la combinación de colores. Por ejemplo:

```html
<iframe
    frameBorder="0"
    height="600"
    class="w-full"
    src="https://widgets.benzinga.com/embed/chart/AAPL?timeframe=6m&exchange=NASDAQ&bgColor=%23ffffff&chartColor=%232962FF"
></iframe>
```

3. **Pega el código en tu sitio web**
   Inserta el código del iframe en la sección deseada de tu HTML o CMS (p. ej., WordPress, Wix).

4. **Prueba la integración**
   Visita tu sitio web para comprobar que el widget se cargue y se muestre correctamente. Ajusta las dimensiones y los estilos si es necesario.
