<div id="creating-a-custom-snippet">
  ## カスタムスニペットの作成
</div>

**前提条件**: スニペットファイルは `snippets` ディレクトリ内に作成する必要があります。

<Note>
  `snippets` ディレクトリ内のページはすべてスニペットとして扱われ、
  単独のページとしてはレンダリングされません。スニペットを単独のページとして作成したい場合は、
  そのスニペットを別のファイルにインポートし、コンポーネントとして
  呼び出してください。
</Note>

<div id="default-export">
  ### デフォルトエクスポート
</div>

1. 複数の箇所で再利用したい内容をスニペットファイルに追加します。
   必要に応じて、スニペットのインポート時に props 経由で
   値を渡せる変数を追加することもできます。

```mdx snippets/my-snippet.mdx
Hello world! This is my content I want to reuse across pages. My keyword of the
day is {word}.
```

<Warning>
  インポートを機能させるには、再利用したいコンテンツを `snippets` ディレクトリ内に配置する必要があります。
</Warning>

2. スニペットを対象ファイルにインポートします。

```mdx destination-file.mdx
---
title: My title
description: My Description
---

import MySnippet from '/snippets/path/to/my-snippet.mdx';

## Header

Lorem impsum dolor sit amet.

<MySnippet word="bananas" />
```

<div id="reusable-variables">
  ### 再利用可能な変数
</div>

1. スニペットファイルで変数をエクスポートします:

```mdx snippets/path/to/custom-variables.mdx
export const myName = 'my name';

export const myObject = { fruit: 'strawberries' };
```

2. 使用先のファイルでスニペットをインポートし、変数を使用します:

```mdx destination-file.mdx
---
title: My title
description: My Description
---

import { myName, myObject } from '/snippets/path/to/custom-variables.mdx';

Hello, my name is {myName} and I like {myObject.fruit}.
```

<div id="reusable-components">
  ### 再利用可能なコンポーネント
</div>

1. スニペットファイル内で、props を受け取るコンポーネントを作成するには、
   アロー関数としてコンポーネントを export します。

```mdx snippets/custom-component.mdx
export const MyComponent = ({ title }) => (
  <div>
    <h1>{title}</h1>
    <p>... snippet content ...</p>
  </div>
);
```

<Warning>
  MDX はアロー関数の本体内ではコンパイルできません。可能であれば HTML
  構文を使用し、MDX を使用する必要がある場合は デフォルトエクスポート を使用してください。
</Warning>

2. スニペットを対象のファイルにインポートし、props を渡します

```mdx destination-file.mdx
---
title: My title
description: My Description
---

import { MyComponent } from '/snippets/custom-component.mdx';

Lorem ipsum dolor sit amet.

<MyComponent title={'Custom title'} />
```
