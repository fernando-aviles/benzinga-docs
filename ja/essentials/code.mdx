<div id="code-block">
  ### コードブロック
</div>

構文ハイライトを有効にするには、コードを 3 つのバッククォートで囲んだ [fenced コードブロック](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) を使用し、先頭のバッククォートの後にスニペットのプログラミング言語を指定します。必要に応じて、プログラミング言語の後にコードの名前を記述することもできます。

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
