<AccordionGroup>
  <Accordion title="Benzinga とは何ですか？また、どのような API を提供していますか？">
    Benzinga は、API を通じて幅広い財務データやニュースサービスを提供する金融テクノロジー企業です。これらの API では、リアルタイムおよび過去の市場データ、ニュース、企業のファンダメンタルズ、取引シグナルなど、さまざまな情報を利用でき、企業は財務データを自社のアプリケーションに直接統合できます。
  </Accordion>

  <Accordion title="Benzinga の API の主な機能は何ですか？">
    Benzinga APIs では、次の機能を利用できます。

    * **リアルタイム市場データ**: 気配値や約定など、ライブデータストリームにアクセスできます。
    * **履歴データ**: 過去の価格や取引量を取得できます。
    * **ニュースとインサイト**: 複数のソースから集約されたニュース、記事、財務インサイトを利用できます。
    * **企業のファンダメンタルズ**: 企業の財務情報、アナリストのレーティング、主要指標を確認できます。
    * **取引シグナルとアラート**: 市場状況に基づくカスタマイズ可能なアラートを利用できます。
  </Accordion>

  <Accordion title="ニュースフィードはどのくらいの頻度で更新されますか？">
    Benzinga のニュースフィード API は、ユーザーが株価の変動や市場動向に関する最新ニュースをタイムリーに受け取れるよう、数分ごとに更新されます。この API には、トップニュース、株価変動の背景、市場アップデートを確認するためのオプションが含まれています。
  </Accordion>

  <Accordion title="Benzinga はリアルタイムデータ向けに WebSocket をサポートしていますか？">
    はい。Benzinga は、リアルタイムデータのストリーミング向けにウェブソケット API を提供しています。この API は、気配値、約定、その他のデータポイントを含む継続的なライブ市場データフィードを必要とするアプリケーションに最適です。ウェブソケット接続は、必要なデータに応じて設定できます。
  </Accordion>

  <Accordion title="どのような履歴データを利用できますか？">
    Benzinga の履歴データ API では、株式、指数、その他の資産を対象に、日次、週次、月次、日中のバーデータなど、さまざまなデータにアクセスできます。利用できる期間は資産や必要な詳細度によって異なりますが、数年前までさかのぼって取得できます。
  </Accordion>

  <Accordion title="Benzinga APIs から決算報告やその他の企業イベントにアクセスできますか？">
    はい。Benzinga は、今後および過去の決算報告、IPO、配当発表、その他の企業イベントを含むカレンダー API を提供しています。これにより、ユーザーは投資に影響を与える重要な財務イベントを把握できます。
  </Accordion>

  <Accordion title="API の利用に制限はありますか？">
    各 API サブスクリプションには、一定時間内に実行できるリクエスト数を定めるレート制限があります。これらの制限は、API の種類やサブスクリプションプランによって異なります。ご利用プランに適用される制限については、Benzinga のサポートチームに問い合わせるか、ドキュメントを参照してください。
  </Accordion>

  <Accordion title="試用版またはサンドボックス環境は利用できますか？">
    Benzinga では、新規ユーザーが有料プランを契約する前に API を試せるよう、試用版またはサンドボックス環境を提供している場合があります。サンドボックスでは、完全版と比べてデータアクセスや機能が制限されることがあります。
  </Accordion>

  <Accordion title="Benzinga は API 統合のためのカスタマーサポートを提供していますか？">
    はい。Benzinga には、API の統合や利用を支援する専任のサポートチームがあります。また、Benzinga API ドキュメントには、ユーザーが API をアプリケーションによりスムーズに統合できるよう、サンプルやコードスニペットも含まれています。
  </Accordion>

  <Accordion title="空売り残高や政府関係者取引などのプレミアムデータには追加料金がかかりますか？">
    空売り残高や政府関係者取引など、一部のデータではプレミアムサブスクリプションまたは追加料金が必要になる場合があります。追加料金の詳細については、Benzinga の営業チームに問い合わせるか、料金に関するドキュメントを確認してください。
  </Accordion>
</AccordionGroup>
