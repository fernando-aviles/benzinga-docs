<AccordionGroup>
  <Accordion title="Benzinga とは何ですか？また、どのような API を提供していますか？">
    Benzinga は、API を通じて幅広い金融データとニュースサービスを提供する金融テクノロジー企業です。これらの API では、リアルタイムおよび過去の市場データ、ニュース、会社のファンダメンタルズ、トレーディングシグナルなど、さまざまな情報を利用でき、企業は金融データを自社のアプリケーションに直接連携できます。
  </Accordion>

  <Accordion title="Benzinga の API の主な機能は何ですか？">
    Benzinga API では、以下を利用できます。

    * **リアルタイム市場データ**: 気配値、約定などを含むライブデータストリームへのアクセス。
    * **過去データ**: 過去の価格や取引量を取得できます。
    * **ニュースとインサイト**: 複数のソースから集約されたニュース、記事、金融インサイト。
    * **会社のファンダメンタルズ**: 企業財務、アナリストのレーティング、主要指標に関する情報。
    * **トレーディングシグナルとアラート**: 市場状況に基づくカスタマイズ可能なアラート。
  </Accordion>

  <Accordion title="ニュースフィードはどのくらいの頻度で更新されますか？">
    Benzinga のニュースフィード API は、ユーザーが株価変動や市場動向に関する最新ニュースをタイムリーに受け取れるよう、数分ごとに更新されます。この API には、注目記事、株価変動の背景、市場の最新情報を表示するためのオプションが含まれています。
  </Accordion>

  <Accordion title="Benzinga はリアルタイムデータ向けに WebSocket をサポートしていますか？">
    はい。Benzinga は、リアルタイムのデータストリーミング向けに WebSocket API を提供しています。この API は、気配値、約定、その他のデータポイントを含む継続的なライブ市場データフィードを必要とするアプリケーションに最適です。WebSocket 接続は、必要なデータに応じて設定できます。
  </Accordion>

  <Accordion title="どのような種類の過去データを利用できますか？">
    Benzinga の過去データ API では、株式、指数、その他の資産を対象に、日次、週次、月次、および日中のバーデータを含むさまざまなデータにアクセスできます。必要な詳細度や資産の種類に応じて、数年分にさかのぼる過去データを利用できます。
  </Accordion>

  <Accordion title="Benzinga API を通じて決算報告やその他の企業イベントにアクセスできますか？">
    はい。Benzinga は、今後および過去の決算報告、IPO、配当発表、その他の企業イベントを含むカレンダーAPIを提供しています。これにより、ユーザーは投資に影響する重要な金融イベントを継続的に把握できます。
  </Accordion>

  <Accordion title="API の利用に制限はありますか？">
    各 API サブスクリプションには、一定期間内に実行できるリクエスト数を定めるレート制限があります。これらの制限は、対象の API とサブスクリプションの階層によって異なります。ご利用プランに適用される制限を確認するには、Benzinga のサポートチームにお問い合わせいただくか、ドキュメントを参照してください。
  </Accordion>

  <Accordion title="試用版またはサンドボックス環境は利用できますか？">
    Benzinga では、新規ユーザーが有料プランを契約する前に API を試せるよう、試用版またはサンドボックス環境を提供している場合があります。サンドボックスでは、完全版と比べて利用できるデータが少なかったり、機能が制限されたりする場合があります。
  </Accordion>

  <Accordion title="Benzinga は API 連携に関するカスタマーサポートを提供していますか？">
    はい。Benzinga には、API の連携や利用を支援する専任のサポートチームがあります。さらに、Benzinga API のドキュメントには、ユーザーが API をアプリケーションへよりスムーズに連携できるよう、サンプルやコードスニペットも含まれています。
  </Accordion>

  <Accordion title="空売り残高や政府関係者の取引などのプレミアムデータには追加料金がかかりますか？">
    空売り残高や政府関係者の取引など、一部のデータでは、プレミアムサブスクリプションまたは追加料金が必要になる場合があります。追加料金の詳細については、Benzinga の営業チームにお問い合わせいただくか、料金に関するドキュメントを確認してください。
  </Accordion>
</AccordionGroup>
