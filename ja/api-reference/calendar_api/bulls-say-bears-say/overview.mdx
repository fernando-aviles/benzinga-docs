<div id="introduction">
  ## はじめに
</div>

Bulls Say Bears Say API は、指定した株式のティッカーシンボルについて、最新の強気および弱気の投資判断の根拠を返します。

* **Bull Cases**: 株式の買いを支持する強気の根拠を示します。
* **Bear Cases**: 株式への投資に反対する弱気の根拠を示します。

このデータは、証券の将来のパフォーマンスに対する相反する見方を把握するうえで有用です。
