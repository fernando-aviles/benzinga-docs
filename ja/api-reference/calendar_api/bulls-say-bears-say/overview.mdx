<div id="introduction">
  ## はじめに
</div>

Bulls Say Bears Say API は、指定した株式ティッカーシンボルについて、最新の強気および弱気の投資シナリオを返します。

* **Bull Cases**: 株式の買いを支持する前向きな根拠を示します。
* **Bear Cases**: 株式への投資に否定的な根拠を示します。

このデータは、当該証券の今後のパフォーマンスに対する相反する見方を理解するのに役立ちます。
