<div id="introduction">
  ## はじめに
</div>

Government Trades API は、米国下院および上院の議員を含む米国政府関係者の取引活動を追跡します。これにより、以下の情報の透明性が確保されます。

* 定期的な取引報告書
* 個別の取引記録
* 資産の種類と取引金額

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-government-trades">
  ### [Government Tradesを取得](/ja/api-reference/calendar_api/government-trades/government-trades)
</div>

連邦議会議員による取引を含む、政府関係者の取引情報を返します。

<div id="get-government-trade-reports">
  ### [政府取引レポートを取得](/ja/api-reference/calendar_api/government-trade-reports/government-trade-reports)
</div>

政府取引の開示レポートの詳細を返します。
