<div id="introduction">
  ## はじめに
</div>

Insider Transactions API は、役員、取締役、大株主などのインサイダーによる株式取引データを提供します。SEC Form 4 の提出情報に基づくデータも含まれます。

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-insider-transaction-filing">
  ### [インサイダー取引届出情報を取得](/ja/api-reference/calendar_api/insider-transaction/insider-transaction-filing)
</div>

株式の購入・売却やオプション行使を含む、インサイダー取引の届出情報を返します。

<div id="get-insider-owner">
  ### [インサイダー所有者を取得](/ja/api-reference/calendar_api/insider-transaction/insider-transaction-owner)
</div>

インサイダー所有者の情報を返します。
