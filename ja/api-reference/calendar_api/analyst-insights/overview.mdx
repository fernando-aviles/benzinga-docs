<div id="introduction">
  ## はじめに
</div>

Analyst Insights API は、金融アナリストによる詳細な分析や推奨を含む、証券に関するリサーチ情報を返します。以下の情報にアクセスできます。

* アナリストによる格上げと格下げ
* 目標株価
* 詳細なリサーチノート
* コンセンサスレーティングに関するコンテキスト情報

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-analyst-insights">
  ### [Analyst Insightsを取得](/ja/api-reference/calendar_api/analyst-insights/get-analyst-insights-v1)
</div>

特定の証券、会社、またはアナリストに関するAnalyst Insightsおよびリサーチ見解を取得します。
