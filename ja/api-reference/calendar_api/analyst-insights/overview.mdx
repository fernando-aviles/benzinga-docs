<div id="introduction">
  ## はじめに
</div>

Analyst Insights API は、金融アナリストによる詳細な分析や推奨を含む、証券に関するリサーチ見解を返します。以下の情報を利用できます。

* アナリストの格上げと格下げ
* 目標株価
* 詳細なリサーチノート
* コンセンサスレーティングに関する情報

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-analyst-insights">
  ### [Analyst Insightsを取得](/ja/api-reference/calendar_api/analyst-insights/get-analyst-insights-v1)
</div>

特定の証券、調査会社、またはアナリストに関するAnalyst Insightsや調査見解を取得します。
