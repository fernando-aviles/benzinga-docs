<div id="introduction">
  ## はじめに
</div>

カレンダー API スイートでは、幅広い財務カレンダーデータにアクセスでき、投資家は重要な企業・経済イベントを追跡できます。

<div id="available-groups">
  ## 利用可能なグループ
</div>

| グループ           | 説明                  | エンドポイント                                                                          |
| :------------- | :------------------ | :------------------------------------------------------------------------------- |
| **カンファレンスコール** | 企業のカンファレンスコールの日程と詳細 | [リンク](/ja/api-reference/calendar_api/conference_calls/returns-conference-call-data) |
| **配当**         | 配当金支払いの履歴と予定        | [リンク](/ja/api-reference/calendar_api/dividends/dividends-events-v22)                |
| **決算**         | 企業の決算発表スケジュール       | [リンク](/ja/api-reference/calendar_api/earnings/returns-the-earnings-data)            |
| **経済指標**       | 世界の経済イベントカレンダー      | [リンク](/ja/api-reference/calendar_api/economics/returns-the-economic-calendar-data)  |
| **ガイダンス**      | 企業の財務ガイダンス          | [リンク](/ja/api-reference/calendar_api/guidance/returns-guidance-data)                |
| **IPO**        | 新規株式公開データ           | [リンク](/ja/api-reference/calendar_api/ipos/ipos-v21)                                 |
| **株式分割**       | 株式分割の履歴と予定          | [リンク](/ja/api-reference/calendar_api/splits/returns-the-splits-calendar-data)       |
| **レーティング**     | 詳細なアナリストレーティング      | [リンク](/ja/api-reference/calendar_api/ratings/returns-analyst-ratings-data)          |
| **M&amp;A**    | 合併・買収の動向            | [リンク](/ja/api-reference/calendar_api/ma/returns-the-mergers-and-acquisitions-data)  |
