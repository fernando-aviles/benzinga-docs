<div id="introduction">
  ## はじめに
</div>

NewsQuantified API は、センチメントスコア、関連性指標、市場への影響指標を含む、定量化されたニュース分析データを返します。ニュース報道の定性的な側面を分析した構造化データを提供しており、以下が含まれます。

* センチメントの極性
* 記事数
* トレンドスコア
* ティッカー固有の指標

このデータは、アルゴリズム取引やクオンツリサーチ向けに設計されています。
