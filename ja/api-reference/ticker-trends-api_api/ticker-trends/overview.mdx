<div id="introduction">
  ## はじめに
</div>

ティッカーのトレンドAPI は、ユーザーエンゲージメント指標に基づき、指定したティッカーシルボルのトレンドデータを取得します。次の項目を追跡することで、関心が高まっている証券の特定に役立ちます。

* トレンドスコア
* アクティビティレベル
* さまざまな時間間隔におけるエンゲージメント

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-ticker-trend-data">
  ### [ティッカーのトレンドデータを取得](/ja/api-reference/ticker-trends-api_api/ticker-trends/get-ticker-trend-data)
</div>

指定したティッカーの集計トレンドスコアを取得します。

<div id="get-trending-tickers-list">
  ### [注目のティッカー一覧を取得](/ja/api-reference/ticker-trends-api_api/ticker-trends/get-ticker-trend-list-data)
</div>

注目度順に並んだティッカーシンボルの一覧を取得します。
