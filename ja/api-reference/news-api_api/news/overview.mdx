<div id="introduction">
  ## はじめに
</div>

Benzinga ニュースフィードAPI は、「Why Is It Moving」 (WIIM) 機能を含む幅広い種類のコンテンツに対応した、リアルタイムの金融ニュースを提供します。以下の項目で柔軟にフィルタリングできます。

* ティッカー、ISIN、CUSIP
* チャンネルとトピック
* 日付範囲と更新タイムスタンプ
* コンテンツ種別と重要度

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-news">
  ### [ニュースを取得](/ja/api-reference/news-api_api/news/get-the-news-items)
</div>

ニュース記事の構造化データを返します。

<div id="get-news-channels">
  ### [ニュースチャンネルを取得](/ja/api-reference/news-api_api/channels/get-available-news-channels)
</div>

フィルタリングに使用できる利用可能なニュースチャンネルの一覧を返します。

<div id="get-removed-news">
  ### [削除済みニュースを取得](/ja/api-reference/news-api_api/news/returns-the-removed-news-data)
</div>

削除または訂正されたニュースを返します。

<div id="why-is-it-moving-wiim">
  ### [Why Is It Moving (WIIM)](/ja/api-reference/news-api_api/news/get-the-news-items)
</div>

妥当な株価変動についての Benzinga 独自の1文解説を利用できます。これらの項目を取得するには、標準のニュースエンドポイントを使用してください。
