<div id="introduction">
  ## はじめに
</div>

バリュエーション比率 API は、証券の相対的な割安・割高度を評価するための主要な指標を提供します。主な比率は次のとおりです。

* P/E (株価収益率)
* P/S (株価売上高倍率)
* P/B (株価純資産倍率)
