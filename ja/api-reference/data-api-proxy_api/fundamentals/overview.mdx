<div id="introduction">
  ## はじめに
</div>

Company ファンダメンタルズAPI は、企業の詳細な分析に必要な包括的な財務データを取得します。対象は次のとおりです。

* **財務諸表**: 貸借対照表、損益計算書、キャッシュフロー計算書 (Original、Restated、TTM) 。
* **企業プロフィール**: 事業内容、セクター、業種、経営陣、本社所在地。
* **決算レポート**: 過去の売上高、EPS、EBITDA、純利益。
* **株式クラスとバリュエーション**: 株式クラスの詳細、アルファ／ベータ、バリュエーション比率。
