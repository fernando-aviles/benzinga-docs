<div id="introduction">
  ## はじめに
</div>

Earning Ratios API は、会社の収益性に関連する各種比率を提供します。たとえば、次のとおりです。

* ROA (総資産利益率)
* ROE (自己資本利益率)
* ROI (投資利益率)
