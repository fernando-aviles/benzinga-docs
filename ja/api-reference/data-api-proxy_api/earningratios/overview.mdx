<div id="introduction">
  ## はじめに
</div>

収益比率 API は、企業の利益や収益性に関する比率を提供します。たとえば、次のような指標があります。

* ROA (総資産利益率)
* ROE (自己資本利益率)
* ROI (投資利益率)
