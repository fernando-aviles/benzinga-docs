<div id="introduction">
  ## はじめに
</div>

Bars API は、指定した証券の過去の OHLCV (始値、高値、安値、終値、出来高) 価格バーデータを取得します。指定した時間足に基づいて集計された価格データを返し、以下をサポートします。

* 複数のティッカー
* さまざまな期間指定 (絶対日付および相対日付)
* カスタマイズ可能な時間バケット (1m、5m、1h、1d など)
* 取引セッションによるフィルタリング (プレマーケット、通常取引、時間外取引)
