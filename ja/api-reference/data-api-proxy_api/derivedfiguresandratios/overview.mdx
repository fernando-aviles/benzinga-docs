<div id="introduction">
  ## はじめに
</div>

Derived Figures and Ratios API は、ファンダメンタルデータ項目に基づいて算出された財務指標を提供します。これには、分析を簡便にするための事前計算済みメトリクスも含まれます。

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-derived-figures">
  ### [派生指標を取得](/ja/api-reference/data-api-proxy_api/derivedfiguresandratios/get-derived-figures-and-ratios-for-a-list-of-symbols)
</div>

シンボルの一覧に対する派生指標と比率を取得します。
