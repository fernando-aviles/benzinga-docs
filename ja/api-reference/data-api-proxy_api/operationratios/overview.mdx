<div id="introduction">
  ## はじめに
</div>

営業比率 API は、企業の経営効率を評価するための指標を提供します。たとえば、次のようなものがあります。

* 総資産回転率
* 棚卸資産回転率
* 売上債権回転率

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-operation-ratios">
  ### [営業比率を取得](/ja/api-reference/data-api-proxy_api/operationratios/get-operation-ratios-v21)
</div>

複数のシンボルの営業比率を取得します。
