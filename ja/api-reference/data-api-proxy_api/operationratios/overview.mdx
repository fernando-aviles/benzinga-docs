<div id="introduction">
  ## はじめに
</div>

営業比率APIは、会社の事業運営の効率性を評価するための指標を提供します。たとえば、次のようなものがあります。

* 総資産回転率
* 棚卸資産回転率
* 売掛金回転率

  ## 利用可能なエンドポイント
</div>

<div id="get-operation-ratios">
  ### [営業比率を取得](/ja/api-reference/data-api-proxy_api/operationratios/get-operation-ratios-v21)
</div>

シンボル一覧の営業比率を取得します。
