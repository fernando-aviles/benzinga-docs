<div id="introduction">
  ## はじめに
</div>

決算説明会トランスクリプト API では、企業の決算説明会に関する全文トランスクリプトと音声ファイルにアクセスできます。内容は次のとおりです。

* 四半期および年次の決算説明会の全文トランスクリプト
* Q&amp;A セッションおよび発言者の識別
* 詳細なナビゲーションに対応するタイムスタンプ
* ダウンロード可能な音声ファイルの URL

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-earnings-call-transcripts">
  ### [決算説明会トランスクリプトを取得](/ja/api-reference/earnings-call-transcripts-api_api/earnings-call-transcripts/get-earnings-call-transcripts)
</div>

指定した企業またはコールIDの決算説明会トランスクリプトを取得します。

<div id="get-earnings-call-audio">
  ### [決算説明会の音声を取得](/ja/api-reference/earnings-call-transcripts-api_api/earnings-call-transcripts/get-earnings-call-transcript-audio-files)
</div>

決算説明会の音声全編をダウンロードするための署名付きURLを取得します。
