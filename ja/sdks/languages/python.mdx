<div id="overview">
  ## 概要
</div>

Benzinga Python SDK は、定量分析や市場インテリジェンスに活用できる、Benzinga の金融データ API およびニュース API への包括的なアクセスを提供します。SDK には金融データ用とニュースデータ用の 2 つの主要モジュールが用意されており、開発者は高度な金融アプリケーションを構築できます。

<div id="installation">
  ## インストール
</div>

SSH 経由で `pip` を使用して SDK をインストールします:

```bash
pip install git+ssh://git@github.com/Benzinga/benzinga-python-client.git
```

<div id="authentication">
  ## 認証
</div>

すべてのSDKクラスで認証にAPIキーが必要です。APIキーを取得するには、[cloud.benzinga.com](https://cloud.benzinga.com)にアクセスしてください。

<div id="modules">
  ## モジュール
</div>

SDKは、主に2つのモジュールで構成されています。

* **財務データ** - 定量的な財務分析と市場データ
* **ニュースデータ** - 金融ニュース、見出し、市場センチメント

***

```python
from benzingaorg import financial_data
```

<div id="price-market-data">
  ### 価格と市場データ
</div>

<div id="price-history">
  #### 価格履歴
</div>

指定した期間の日次の過去ローソク足データを取得します。

```python
data = fin.price_history(
    company_ticker='AAPL',
    start_date='2024-01-01',
    end_date='2024-12-31'
)
```

**パラメータ:**

* `company_ticker` (必須) - 株式ティッカー
* `start_date` (必須) - YYYY-MM-DD形式の開始日
* `end_date` (必須) - YYYY-MM-DD形式の終了日

<div id="quote">
  #### クオート
</div>

OHLC、売気配/買気配、出来高、騰落率を含むリアルタイムのクオートデータを取得します。

```python
quote = fin.quote(company_ticker='AAPL')
```

**戻り値:** シンボル、取引所、種別、名称、始値、高値、安値、終値、買い気配値、売り気配値、出来高、騰落率

<div id="chart">
  #### チャート
</div>

時間間隔を指定してチャートデータを取得します。

```python
chart = fin.chart(
    company_ticker='AAPL',
    start_date='2024-01-01',
    end_date='2024-12-31',
    interval='1D'  # オプション: 1MONTH, 1W, 1D, 1H, 15M
)
```

**間隔オプション:** `1MONTH`, `1W`, `1D`, `1H`, `15M`

<div id="news-methods">
  ### ニュースメソッド
</div>

<div id="news">
  #### ニュース
</div>

柔軟なフィルタリング条件と表示オプションでニュース記事を取得します。

```python
articles = news.news(
    pagesize=15,              # デフォルト: 15
    display_output='full',    # オプション: full, abstract, headline
    date_from='2024-01-01',   # 任意
    date_to='2024-12-31',     # 任意
    channel=None              # 任意: チャンネルでフィルタリング
)
```

**表示オプション:**

* `full` - 記事の全文
* `abstract` - 記事の要約
* `headline` - 見出しのみ

<div id="top-news">
  #### トップニュース
</div>

```python
top = news.top_news(

    channel=None,   # チャンネルフィルター（省略可）
    limit=10        # 返す記事の件数
)
```

<div id="output-formatting">
  ## 出力形式
</div>

どちらのモジュールも、JSON を見やすく表示するための `.output()` メソッドをサポートしています。

```python
# 財務データの出力
data = fin.quote('AAPL')
fin.output(data)

# ニュースデータの出力
articles = news.news(pagesize=10)
news.output(articles)
```

<div id="complete-example">
  ## サンプル全体
</div>

```python
from benzingaorg import financial_data, news_data

# モジュールの初期化
fin = financial_data.Benzinga(api_key='YOUR_API_KEY')
news = news_data.News(api_key='YOUR_API_KEY')

# 財務データの取得
quote = fin.quote('AAPL')
fundamentals = fin.fundamentals('AAPL')
earnings = fin.earnings(ticker='AAPL', date_from='2024-01-01')

# ニュースデータの取得
articles = news.news(pagesize=20, display_output='full')
top_news = news.top_news(limit=5)

# フォーマットと表示
fin.output(quote)
news.output(articles)
```
