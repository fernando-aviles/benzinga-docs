<div id="overview">
  ## 概要
</div>

Benzinga Python SDK は、定量分析や市場インテリジェンス向けに、Benzinga の金融データ API とニュース API へ包括的にアクセスするための機能を提供します。この SDK には、金融データ用とニュースデータ用の 2 つの主要モジュールがあり、開発者は高度な金融アプリケーションを構築できます。

<div id="installation">
  ## インストール
</div>

SSH 経由で `pip` を使用して SDK をインストールします:

```bash
pip install git+ssh://git@github.com/Benzinga/benzinga-python-client.git
```

<div id="authentication">
  ## 認証
</div>

すべてのSDKクラスで、認証にAPIキーが必要です。APIキーを取得するには、[cloud.benzinga.com](https://cloud.benzinga.com) にアクセスしてください。

<div id="modules">
  ## モジュール
</div>

このSDKは、主に次の2つのモジュールで構成されています。

* **金融データ** - 定量的な財務分析および市場データ
* **ニュースデータ** - 金融ニュース、見出し、マーケットセンチメント

***

  ## 金融データモジュール

```python
from benzingaorg import financial_data

fin = financial_data.Benzinga(api_key='YOUR_API_KEY')
```

<div id="price-market-data">
  ### 価格・市場データ
</div>

<div id="price-history">
  #### 価格履歴
</div>

指定した日付範囲の日次ローソク足データを取得します。

```python
data = fin.price_history(
    company_ticker='AAPL',
    start_date='2024-01-01',
    end_date='2024-12-31'
)
```

**パラメータ:**

* `company_ticker` (required) - 株式ティッカーシンボル
* `start_date` (required) - YYYY-MM-DD 形式の開始日
* `end_date` (required) - YYYY-MM-DD 形式の終了日

<div id="quote">
  #### Quote
</div>

OHLC、買値/売値、出来高、騰落率を含むリアルタイムのクオートデータを取得します。

```python
quote = fin.quote(company_ticker='AAPL')
```

**戻り値:** シンボル、取引所、type、銘柄名、始値、高値、安値、終値、買気配値、売気配値、出来高、騰落率

<div id="chart">
  #### チャート
</div>

任意の時間間隔を指定してチャートデータを取得します。

```python
chart = fin.chart(
    company_ticker='AAPL',
    start_date='2024-01-01',
    end_date='2024-12-31',
    interval='1D'  # 省略可能: 1MONTH, 1W, 1D, 1H, 15M
)
```

**時間間隔オプション:** `1MONTH`, `1W`, `1D`, `1H`, `15M`

<div id="news-methods">
  ### ニュース関連メソッド
</div>

<div id="news">
  #### ニュース
</div>

柔軟なフィルタ条件と表示オプションを指定して、ニュース記事を取得します。

```python
articles = news.news(
    pagesize=15,              # デフォルト: 15
    display_output='full',    # オプション: full, abstract, headline
    date_from='2024-01-01',   # 省略可
    date_to='2024-12-31',     # 省略可
    channel=None              # 省略可: チャンネルでフィルタリング
)
```

**表示オプション:**

* `full` - 記事の全文
* `abstract` - 記事の要約
* `headline` - 見出しのみ

<div id="top-news">
  #### トップニュース
</div>

話題のニュース記事を取得します。

```python
top = news.top_news(
    type=None,      # ニュースタイプ（省略可）
    channel=None,   # チャンネルフィルター（省略可）
    limit=10        # 返す記事数
)
```

<div id="output-formatting">
  ## 出力形式
</div>

どちらのモジュールも、JSON を見やすく表示するための `.output()` メソッドをサポートしています。

```python
# 金融データの出力
data = fin.quote('AAPL')
fin.output(data)

# ニュースデータの出力
articles = news.news(pagesize=10)
news.output(articles)
```

***

<div id="complete-example">
  ## 完全なサンプル
</div>

```python
from benzingaorg import financial_data, news_data

# モジュールの初期化
fin = financial_data.Benzinga(api_key='YOUR_API_KEY')
news = news_data.News(api_key='YOUR_API_KEY')

# 金融データの取得
quote = fin.quote('AAPL')
fundamentals = fin.fundamentals('AAPL')
earnings = fin.earnings(ticker='AAPL', date_from='2024-01-01')

# ニュースデータの取得
articles = news.news(pagesize=20, display_output='full')
top_news = news.top_news(limit=5)

# フォーマットと表示
fin.output(quote)
news.output(articles)
```
