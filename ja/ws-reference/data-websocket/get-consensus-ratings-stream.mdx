<div id="overview">
  ## 概要
</div>

複数のアナリストの見解を集約した、統合レーティングと目標株価のリアルタイムなコンセンサスレーティングを購読できます。市場全体のセンチメントや、アナリストコンセンサスの変化を追跡できます。

<div id="key-features">
  ### 主な機能
</div>

* **コンセンサス集計**: 銘柄に対するすべてのアナリストレーティングを統合した表示
* **目標株価コンセンサス**: 目標株価の平均値、高値、安値
* **レーティング分布**: Strong Buy、Buy、Hold、Sell、Strong Sell ごとの内訳
* **アナリスト数**: 総アナリスト数とユニークなアナリスト数を追跡
* **柔軟な集計**: 件数または割合で表示

<div id="use-cases">
  ### ユースケース
</div>

* コンセンサスレーティングのdashboardやウィジェットを構築する
* アナリスト全体のセンチメントの変化を追跡する
* 個別のレーティングをコンセンサスと比較する
* 目標株価のレンジ (上限、下限、平均) を監視する
* コンセンサスが改善または悪化している銘柄を特定する

<div id="quick-start">
  ## クイックスタート
</div>

上記の対話型 WebSocket テスターを使用して接続し、ストリームをリアルタイムでテストします。

<div id="connection-url">
  ### 接続 URL
</div>

```
wss://api.benzinga.com/api/v1/consensus-ratings/stream?token=YOUR_TOKEN
```

<div id="query-parameters">
  ### クエリパラメータ
</div>

| パラメータ            | 必須  | 説明                                                 |
| ---------------- | --- | -------------------------------------------------- |
| `token`          | はい  | Benzinga WebSocket API の token (bz.production***)  |
| `tickers`        | いいえ | ティッカーのカンマ区切りリスト (例: `AAPL,MSFT`)                   |
| `isins`          | いいえ | 絞り込み対象の ISIN のカンマ区切りリスト                            |
| `simplify`       | いいえ | レーティングを Buy、Hold、Sell に簡略化                         |
| `aggregate_type` | いいえ | 集計方法: `number` または `percentage`                    |

<div id="consensus-ratings">
  ### コンセンサスレーティング
</div>

| レーティング        | 数値        | 説明    |
| ------------- | --------- | ----- |
| `STRONG_BUY`  | 5.0       | 非常に強気 |
| `BUY`         | 4.0 - 4.9 | 強気    |
| `HOLD`        | 3.0 - 3.9 | 中立    |
| `SELL`        | 2.0 - 2.9 | 弱気    |
| `STRONG_SELL` | 1.0 - 1.9 | 非常に弱気 |

<div id="key-metrics">
  ### 主要指標
</div>

| フィールド                    | 説明                        |
| ------------------------ | ------------------------- |
| `consensus_rating`       | 総合レーティング (BUY、HOLD、SELL)  |
| `consensus_rating_val`   | レーティングの数値 (1.0～5.0)       |
| `consensus_price_target` | アナリスト全体の平均目標株価            |
| `high_price_target`      | 最高目標株価                    |
| `low_price_target`       | 最低目標株価                    |
| `total_analyst_count`    | レーティングの総件数                |
| `unique_analyst_count`   | 重複を除いたアナリスト数              |

<div id="interactive-commands">
  ## インタラクティブコマンド
</div>

| Command  | Description                   |
| -------- | ----------------------------- |
| `ping`   | 接続を維持します (`pong` が返されます)      |
| `replay` | 直近のキャッシュ済みメッセージを最大100件まで再送します |

<div id="best-practices">
  ## ベストプラクティス
</div>

* **コンセンサスの変化**: `consensus_rating_val` の推移を継続的に監視します
* **目標株価レンジ**: バリュエーションの参考として、現在価格を目標株価の上限値・下限値と比較します
* **カバレッジの厚み**: アナリスト数が多いほど、コンセンサスの信頼性が高い傾向があります
* **分布分析**: センチメントの違いをより詳しく把握するため、`aggregate_ratings` の内訳を確認します
* **割合表示**: 正規化して比較するには、`aggregate_type=percentage` を使用します
