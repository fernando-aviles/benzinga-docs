<div id="overview">
  ## 概要
</div>

リアルタイムのアナリストによるレーティング変更や目標株価の更新を購読できます。格上げ、格下げ、新規評価の開始、目標株価の調整を発生と同時に監視できます。

<div id="key-features">
  ### 主な機能
</div>

* **リアルタイムレーティング**: アナリストのレーティング変更を即座に通知
* **目標株価**: 調整履歴を含む現在および過去の目標株価を追跡
* **アナリストの詳細**: アナリスト名、調査会社、過去の精度指標を含む
* **アクション分類**: 格上げ、格下げ、据え置き、新規付与、再表明
* **銘柄で絞り込み**: 特定のティッカーまたはISINをサブスクライブ

<div id="use-cases">
  ### ユースケース
</div>

* リアルタイムのレーティング変更アラートや通知を作成
* アナリストや調査会社の実績と精度を追跡
* 目標株価コンセンサスの変化を監視
* レーティングの格上げ/格下げダッシュボードを作成
* アナリストの評価変更に基づく取引戦略を自動化

<div id="quick-start">
  ## クイックスタート
</div>

上部のインタラクティブなウェブソケットテスターを使用して接続し、ストリームをリアルタイムでテストします。

<div id="connection-url">
  ### 接続URL
</div>

```
wss://api.benzinga.com/api/v2.1/calendar/ratings/stream?token=YOUR_TOKEN
```

<div id="query-parameters">
  ### クエリパラメータ
</div>

| パラメータ     | 必須  | 説明                                             |
| --------- | --- | ---------------------------------------------- |
| `token`   | はい  | Benzinga のウェブソケット API トークン (bz.production***)  |
| `tickers` | いいえ | ティッカーのカンマ区切りリスト (例: `AAPL,TSLA`)               |
| `isins`   | いいえ | フィルタリングする ISIN のカンマ区切りリスト                      |

<div id="rating-actions">
  ### レーティングアクション
</div>

| アクション        | 説明                                 |
| ------------ | ---------------------------------- |
| `Upgrades`   | レーティングが引き上げられた (例: Neutral → Buy)  |
| `Downgrades` | レーティングが引き下げられた (例: Buy → Hold)     |
| `Maintains`  | レーティングに変更はない                       |
| `Initiates`  | 初回のレーティング付与                        |
| `Reiterates` | 既存のレーティングを維持することを確認                |

<div id="message-actions">
  ### メッセージアクション
</div>

| アクション                 | 説明          |
| --------------------- | ----------- |
| `created` / `Created` | 新しいレーティングが公開された |
| `updated` / `Updated` | レーティングが更新された    |
| `deleted` / `Deleted` | レーティングが削除された    |

<div id="interactive-commands">
  ## 対話コマンド
</div>

| Command  | Description                   |
| -------- | ----------------------------- |
| `ping`   | 接続を維持します (`pong` が返されます)      |
| `replay` | 直近のキャッシュ済みメッセージを最大100件まで再送します |

<div id="best-practices">
  ## ベストプラクティス
</div>

* **変更の追跡**: 格上げ/格下げを検出するには、`rating_current` と `rating_prior` を比較します
* **目標株価の変動**: 目標株価の変更を把握するため、`pt_current` と `pt_prior` を監視します
* **アナリストの信頼性**: 調査会社の重要度と過去の的中精度を考慮します
* **重要度フィルター**: 影響の大きい高重要度 (4～5) のレーティングに注目します
