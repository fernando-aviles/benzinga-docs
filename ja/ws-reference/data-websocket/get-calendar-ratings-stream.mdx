<div id="overview">
  ## 概要
</div>

アナリストレーティングの変更や目標株価の更新をリアルタイムで購読できます。格上げ、格下げ、カバレッジ開始、目標株価の調整を、発生した時点でリアルタイムに監視できます。

<div id="key-features">
  ### 主な機能
</div>

* **リアルタイムのレーティング**: アナリストのレーティング変更を即座に通知で受け取れます
* **目標株価**: 調整内容を含む現在および過去の目標株価を追跡できます
* **アナリスト詳細**: アナリスト名、所属企業、過去の的中率指標を含みます
* **アクション分類**: 格上げ、格下げ、据え置き、新規付与、再表明
* **証券で絞り込み**: 特定のティッカーまたはISINを指定して購読できます

<div id="use-cases">
  ### ユースケース
</div>

* リアルタイムのレーティング変更アラートと通知を構築
* アナリスト会社の実績と精度を追跡
* 目標株価コンセンサスの変化を監視
* レーティングの格上げ/格下げダッシュボードを作成
* アナリストの評価変更に基づく取引戦略を自動化

<div id="quick-start">
  ## クイックスタート
</div>

上記の対話型 WebSocket テスターを使用して接続し、ストリームをリアルタイムでテストできます。

<div id="connection-url">
  ### 接続 URL
</div>

```
wss://api.benzinga.com/api/v2.1/calendar/ratings/stream?token=YOUR_TOKEN
```

<div id="query-parameters">
  ### クエリパラメータ
</div>

| パラメータ     | 必須  | 説明                                             |
| --------- | --- | ---------------------------------------------- |
| `token`   | はい  | Benzinga WebSocket APIトークン (bz.production***)  |
| `tickers` | いいえ | ティッカーシンボルのカンマ区切りリスト (例: `AAPL,TSLA`)           |
| `isins`   | いいえ | 絞り込み用のISINのカンマ区切りリスト                           |

<div id="rating-actions">
  ### レーティングアクション
</div>

| アクション        | 説明                             |
| ------------ | ------------------------------ |
| `Upgrades`   | レーティング格上げ (例: Neutral → Buy)  |
| `Downgrades` | レーティング格下げ (例: Buy → Hold)     |
| `Maintains`  | レーティングに変更なし                    |
| `Initiates`  | 初回レーティング付与                     |
| `Reiterates` | 既存レーティングの再表明                   |

<div id="message-actions">
  ### メッセージアクション
</div>

| アクション                 | 説明                |
| --------------------- | ----------------- |
| `created` / `Created` | 新しいレーティングが公開されました |
| `updated` / `Updated` | レーティングが変更されました    |
| `deleted` / `Deleted` | レーティングが削除されました    |

<div id="interactive-commands">
  ## インタラクティブコマンド
</div>

| Command  | Description                    |
| -------- | ------------------------------ |
| `ping`   | 接続を維持します (`pong` を返します)        |
| `replay` | キャッシュされた直近 100 件までのメッセージを再送します |

<div id="best-practices">
  ## ベストプラクティス
</div>

* **変更の追跡**: アップグレード/ダウングレードを検出するには、`rating_current` と `rating_prior` を比較します
* **目標株価の変動**: 目標株価の変更を把握するには、`pt_current` と `pt_prior` を監視します
* **アナリストの信頼性**: 分析機関の重要度と過去の的中精度を考慮します
* **重要度フィルター**: 影響の大きいレーティングに注目するには、重要度の高い (4-5) レーティングに絞り込みます
