<div id="overview">
  ## 概要
</div>

リアルタイムのアナリストインサイトを購読すると、アナリストのレーティング、目標株価、詳細な推奨に関する更新をリアルタイムで受け取れます。

<div id="key-features">
  ### 主な機能
</div>

* **リアルタイム更新**: アナリストが新しいインサイトを公開したり、既存のレーティングを更新した際に、即座に通知を受け取れます
* **銘柄で絞り込み**: 特定のティッカーまたはISINを購読して、関連するデータのみを受信できます
* **包括的なデータ**: アナリスト会社の詳細、レーティング、目標株価、詳細なインサイトが含まれます
* **変更の追跡**: インサイトが作成、更新、削除されたタイミングを追跡できます

<div id="use-cases">
  ### 利用例
</div>

* ポートフォリオ内の証券に対するアナリストの見方の変化を追跡する
* レーティングの格上げや格下げに対するリアルタイムアラートを作成する
* 複数のアナリストによる目標株価の変更を監視する
* 複数のアナリスト会社からコンセンサスの見解を集約する

<div id="quick-start">
  ## クイックスタート
</div>

上部の対話型WebSocketテスターを使用して接続し、ストリームをリアルタイムでテストできます。

<div id="connection-url">
  ### 接続 URL
</div>

```
wss://api.benzinga.com/api/v1/analyst/insights/stream?token=YOUR_TOKEN
```

<div id="query-parameters">
  ### クエリパラメータ
</div>

| パラメータ     | 必須  | 説明                                                 |
| --------- | --- | -------------------------------------------------- |
| `token`   | はい  | Benzinga WebSocket API の token (bz.production***)  |
| `tickers` | いいえ | ティッカーシンボルのカンマ区切りリスト (例: `AAPL,MSFT`)               |
| `isins`   | いいえ | 絞り込み対象の ISIN のカンマ区切りリスト                            |

</div>


<div id="message-actions">
  ### メッセージアクション
</div>

| アクション                 | 説明                  |
| --------------------- | ------------------- |
| `created` / `Created` | 新しいアナリストインサイトが公開された |
| `updated` / `Updated` | 既存のインサイトが更新された      |
| `deleted` / `Deleted` | インサイトが削除された         |

<div id="interactive-commands">
  ## 対話型コマンド
</div>

| コマンド     | 説明                           |
| -------- | ---------------------------- |
| `ping`   | 接続を維持します (`pong` が返されます)     |
| `replay` | キャッシュされた直近100件までのメッセージを再送します |

<div id="best-practices">
  ## ベストプラクティス
</div>

* **重複排除**: 重複メッセージの処理を避けるために `id` フィールドを使用します
* **ハートビート**: 接続を維持するため、30〜60秒ごとに `ping` コマンドを送信します
* **エラー処理**: 指数バックオフを用いた再接続ロジックを実装します
* **フィルタリング**: 帯域幅を削減するために、ティッカー/ISIN フィルターを使用します
