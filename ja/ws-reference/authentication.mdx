---
title: "認証"
description: "Benzinga API へのリクエストを認証する方法。"
---

Benzinga API にアクセスするには、各リクエストを API tokenで認証する必要があります。API tokenは、[Benzinga Console](https://www.api.benzinga.com/token) から生成および管理できます。

> **重要:** API tokenを使うとお使いの Benzinga アカウントにアクセスできるため、厳重に管理してください。公開リポジトリ、クライアントサイドのコード、その他の安全でない場所で共有しないでください。

<div id="authenticating-requests">
  ## リクエストの認証
</div>

Benzinga API では **Bearer Authentication** を使用します。これは、各リクエストの `token` クエリパラメータに token を含めるセキュアな認証方式です。この token は、Benzinga リソースへのアクセス権限を示す bearer として機能します。

<div id="example-request-with-bearer-token">
  ### Bearer Token を使用したリクエスト例
</div>

次のように、リクエストの `Token` クエリパラメータにtokenを含めます。

```bash
Token: <token>
```
