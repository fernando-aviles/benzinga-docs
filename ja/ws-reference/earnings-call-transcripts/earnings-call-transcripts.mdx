<div id="websocket-websocketv1">
  ## WebSocket `websocket/v1`:
</div>

```txt
wss://api.benzinga.com/api/v1/earnings-call-transcripts/stream
```

<div id="parameters">
  ### パラメータ:
</div>

| 名前        |  場所 |    型   | 必須 |                                             説明 |
| :-------- | :-: | :----: | -: | ---------------------------------------------: |
| `token`   | クエリ | string | はい | Benzinga WebSocket用のtoken (bz.production***) 。 |
| `tickers` | クエリ | string | 任意 |                          購読したいティッカーの一覧。カンマ区切り。 |

<div id="stream-response">
  ### ストリームレスポンス:
</div>

ストリームを購読すると、以下の形式でカンファレンスコールのトランスクリプトに関する更新を受信します。

```json
{
   "api_version":"EarningsCallTranscripts/v1",
   "kind":"stream_type",
   "data":{
      "type":"transcription_end",
      "content":{
         "call_id":"6757fbd6f0cc56000183c6a5",
         "transcript_chunk":"We are extremely pleased with our fourth quarter and full fiscal year 2024 results.",
         "security":{
            "company_name":"Limoneira",
            "ticker":"LMNR",
            "cik":"1342423",
            "isin":"US5327461043",
            "exchange":"NAS"
         },
         "time":1735189664,
         "index":0
      }
   }
}
```

<div id="responses">
  #### レスポンス
</div>

* **200 OK**
* **説明**: 成功
* **コンテンツタイプ**: `application/json`
* **スキーマ**: [EarningsCallTranscriptsWSResp](#earningscalltranscriptswsresp)

<div id="earningscalltranscriptswsresp">
  ### EarningsCallTranscriptsWSResp
</div>

| Property        | Type   | Description                                                |
| --------------- | ------ | ---------------------------------------------------------- |
| api&#95;version | string | WebSocket API のバージョン: `EarningsCallTranscripts/v1`。        |
| kind            | string | WebSocket 接続ストリームのタイプ。                                     |
| data            | object | WebSocket データの詳細を含むオブジェクト。                                 |
| - type          | string | メッセージの種類 (例: transcript&#95;chunk、transcription&#95;end) 。 |
| - content       | object | `EarningsCallTranscripts` スキーマへの参照。                        |

<div id="earningscalltranscripts">
  ### EarningsCallTranscripts
</div>

| Property             | Type    | Description                                             |
| -------------------- | ------- | ------------------------------------------------------- |
| call&#95;id          | string  | 一意のコールID。                                               |
| transcript&#95;chunk | string  | カンファレンスコールの文字起こしチャンク。                                   |
| security             | object  | インサイトが提供される対象の証券情報を含みます。                                |
| - cik                | string  | 証券の CIK (Central Index Key) 。                           |
| - exchange           | string  | 証券が上場されている取引所。                                          |
| - isin               | string  | ISIN (International Securities Identification Number) 。 |
| - name               | string  | 証券名。                                                    |
| - symbol             | string  | 証券のティッカーシンボル。                                           |
| time                 | integer | 文字起こしチャンクの作成時刻のタイムスタンプ (float64 形式) 。                   |
| index                | integer | 文字起こしチャンクのインデックス (int 形式) 。                             |
