<div id="websocket-websocketv1">
  ## ウェブソケット `websocket/v1`:
</div>

```txt
wss://api.benzinga.com/api/v1/earnings-call-transcripts/stream
```

<div id="parameters">
  ### パラメータ:
</div>

| 名前        |  場所 |  型  | 必須 |                                          説明 |
| :-------- | :-: | :-: | -: | ------------------------------------------: |
| `token`   | クエリ | 文字列 | 必須 | Benzinga ウェブソケットのtoken (bz.production***) 。 |
| `tickers` | クエリ | 文字列 | 任意 |                購読するティッカーの一覧です。カンマ区切りで指定します。 |

<div id="stream-response">
  ### ストリームレスポンス:
</div>

ストリームを購読すると、カンファレンスコールのトランスクリプトに関する更新が次の形式で配信されます:

```json
{
   "api_version":"EarningsCallTranscripts/v1",
   "kind":"stream_type",
   "data":{
      "type":"transcription_end",
      "content":{
         "call_id":"6757fbd6f0cc56000183c6a5",
         "transcript_chunk":"We are extremely pleased with our fourth quarter and full fiscal year 2024 results.",
         "security":{
            "company_name":"Limoneira",
            "ticker":"LMNR",
            "cik":"1342423",
            "isin":"US5327461043",
            "exchange":"NAS"
         },
         "time":1735189664,
         "index":0
      }
   }
}
```

<div id="responses">
  #### レスポンス
</div>

* **200 OK**
* **説明**: 正常終了
* **コンテンツタイプ**: `application/json`
* **スキーマ**: [EarningsCallTranscriptsWSResp](#earningscalltranscriptswsresp)

<div id="earningscalltranscriptswsresp">
  ### EarningsCallTranscriptsWSResp
</div>

| Property        | Type   | Description                                                |
| --------------- | ------ | ---------------------------------------------------------- |
| api&#95;version | string | ウェブソケット API のバージョン: `EarningsCallTranscripts/v1`。          |
| kind            | string | ウェブソケット接続ストリームの種別。                                         |
| data            | object | ウェブソケットデータの詳細を含みます。                                        |
| - type          | string | メッセージの種別 (例: transcript&#95;chunk、transcription&#95;end) 。 |
| - content       | object | `EarningsCallTranscripts` スキーマへの参照。                        |

<div id="earningscalltranscripts">
  ### EarningsCallTranscripts
</div>

| Property             | Type    | Description                                  |
| -------------------- | ------- | -------------------------------------------- |
| call&#95;id          | string  | 一意のコールID。                                    |
| transcript&#95;chunk | string  | カンファレンスコールのトランスクリプトの一部。                      |
| security             | object  | 対象証券の詳細を含みます。                                |
| - cik                | string  | 証券のCIK。                                      |
| - exchange           | string  | 証券が上場している取引所。                                |
| - isin               | string  | ISIN (国際証券識別番号) 。                            |
| - name               | string  | 証券の名称。                                       |
| - symbol             | string  | 証券のティッカーシンボル。                                |
| time                 | integer | トランスクリプトチャンクが作成された時刻のタイムスタンプ (形式: float64) 。 |
| index                | integer | トランスクリプトチャンクのインデックス (形式: int) 。              |
