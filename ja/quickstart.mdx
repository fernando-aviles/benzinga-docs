<div id="setup-your-development">
  ## 開発環境のセットアップ
</div>

ローカルでドキュメントを更新し、公開環境にデプロイする方法を説明します。

<div id="edit-and-preview">
  ### 編集とプレビュー
</div>

<AccordionGroup>
  <Accordion icon="github" title="ドキュメントをローカルにクローンする">
    オンボーディングの過程で、GitHub 上に
    お客様のドキュメントを含むリポジトリを作成しました。このリポジトリは
    当社の
    [ダッシュボード](https://dashboard.mintlify.com)で確認できます。リポジトリを
    ローカルにクローンするには、ターミナルで以下の
    [手順](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)
    に従ってください。
  </Accordion>

  <Accordion icon="rectangle-terminal" title="変更をプレビューする">
    プレビューすると、変更が意図したとおりに表示されるかを確認できます。これらの変更をローカルでレンダリングするための
    コマンドラインインターフェースを用意しています。1. 次のコマンドを使って
    [Mintlify CLI](https://www.npmjs.com/package/mintlify) をインストールし、ドキュメントの変更をローカルでプレビューします: `npm i -g mintlify`
    2. ドキュメントのルート (
    `docs.json` がある場所) で次のコマンドを実行します: `mintlify dev`
  </Accordion>
</AccordionGroup>

<div id="deploy-your-changes">
  ### 変更をデプロイする
</div>

<AccordionGroup>
  <Accordion icon="message-bot" title="GitHub アプリをインストールする">
    当社の GitHub アプリは、変更内容をドキュメントサイトに自動的にデプロイするため、
    デプロイを自分で管理する必要はありません。インストール用のリンクは
    [ダッシュボード](https://dashboard.mintlify.com) で確認できます。ボットの
    インストールが正常に完了すると、リポジトリのコミットハッシュの横に
    チェックマークが表示されます。
  </Accordion>

  <Accordion icon="rocket" title="変更をプッシュする">
    ドキュメントサイトに変更を反映するには、
    [変更をコミットして Git にプッシュして
    ください](https://docs.github.com/en/get-started/using-git/pushing-commits-to-a-remote-repository#about-git-push)。
    プッシュしても GitHub アプリによる変更のデプロイが正常に完了したことを確認できない
    場合は、[ダッシュボード](https://dashboard.mintlify.com) から手動で
    ドキュメントを更新することもできます。
  </Accordion>
</AccordionGroup>

<div id="update-your-docs">
  ## ドキュメントを更新
</div>

MDX 構文と React コンポーネントを使って、ファイルに直接コンテンツを追加できます。用意されているコンポーネントはどれでも使用でき、独自のコンポーネントを作成することもできます。

<CardGroup>
  <Card title="ドキュメントをカスタマイズ" icon="paintbrush" href="/ja/settings/global">
    ブランドに合わせた設定で、ドキュメントの見た目を整えましょう。
  </Card>

  <Card title="API エンドポイントを追加" icon="square-code" href="/ja/api-playground/configuration">
    OpenAPI 仕様を実装して、API を操作できるようにします。
  </Card>

  <Card title="アナリティクスを統合" icon="chart-mixed" href="/ja/analytics/supported-integrations">
    ドキュメント上でのユーザー操作からインサイトを得られます。
  </Card>

  <Card title="カスタムドメインでホスト" icon="browser" href="/ja/settings/custom-domain/subdomain">
    自社サイトのサブドメインでドキュメントを公開できます。
  </Card>
</CardGroup>
