</div>

**新機能**

**改善**

* ニュースのウェブソケットペイロードに画像 URL を追加

* Government Trade API の更新に対応
* カンファレンスコールカレンダーに period/year データをバックフィル

**バグ修正**

* CIBC の動的ウィジェットに関する問題を修正
* 単一 date クエリで Consensus Ratings エンドポイントがゼロを返す問題を解消
* /sync エンドポイントの重複排除ロジックを修正
* Morgan Stanley の優先株ロゴ欠落を調査
