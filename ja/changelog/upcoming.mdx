<div id="platform-services">
  ### Platform Services
</div>

**新機能**

* より適切な分類のため、「Conference Call Topics」辞書を追加
* composite&#95;auto の SVG 画像に対応した Logos V2 サポート
* Halts/Resumes Calendar API の API 仕様を追加
* 二重反転ロジック向けの「Hedge Message」を実装
* すべてのサービスでレート制限とメッセージングを改善
* 接続監視用の Corologix アラートを追加

**改善**

* S3 ニュース配信の監視を追加
* 従来の TDA「Benzinga Attribution」フィールドを削除
* Analyst Dictionary に「Active」フィールドを追加 (Yes/No オプション) 
* すべての SoFi エンドポイントで ISIN クエリをサポート
* News WebSocket ペイロードに画像 URL を追加
* 株式分割に伴う JPMC のアップサイド計算の問題を解消
* Government Trade API の更新に対応
* カンファレンスコールカレンダーに期間/年データを補完

**バグ修正**

* CIBC の動的ウィジェットに関する問題を修正
* 単一日付クエリで Consensus Ratings エンドポイントがゼロを返す問題を解消
* `/sync` エンドポイントの重複排除ロジックを修正
* Morgan Stanley の優先株ロゴ欠落を調査
