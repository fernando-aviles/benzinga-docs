<div id="setup-your-development">
  ## 開発環境をセットアップする
</div>

ローカルでドキュメントを更新し、一般公開する方法を学びます。

<div id="edit-and-preview">
  ### 編集とプレビュー
</div>

<AccordionGroup>
  <Accordion icon="github" title="ドキュメントをローカルにクローンする">
    オンボーディング時に、GitHub 上にドキュメントコンテンツを含む
    リポジトリを作成しました。このリポジトリは
    [dashboard](https://dashboard.mintlify.com)で確認できます。リポジトリを
    ローカルにクローンするには、ターミナルで以下の
    [手順](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)
    に従ってください。
  </Accordion>

  <Accordion icon="rectangle-terminal" title="変更をプレビューする">
    プレビューすると、変更が意図したとおりに表示されるかを確認できます。これらの
    変更をローカルでレンダリングするためのコマンドラインインターフェースを用意しています。1.
    次のコマンドで
    [Mintlify CLI](https://www.npmjs.com/package/mintlify)をインストールすると、ドキュメントの
    変更をローカルでプレビューできます: `npm i -g mintlify`
    2. ドキュメントのルート (
    `docs.json` がある場所) で、次のコマンドを実行します: `mintlify dev`
  </Accordion>
</AccordionGroup>

<div id="deploy-your-changes">
  ### 変更をデプロイする
</div>

<AccordionGroup>
  <Accordion icon="message-bot" title="GitHub アプリをインストールする">
    変更内容は GitHub アプリによってドキュメントサイトに自動でデプロイされるため、
    デプロイを自分で管理する必要はありません。インストール用のリンクは
    [dashboard](https://dashboard.mintlify.com)で確認できます。ボットの
    インストールが正常に完了すると、リポジトリのコミットハッシュの横に
    チェックマークが表示されます。
  </Accordion>

  <Accordion icon="rocket" title="変更をプッシュする">
    ドキュメントサイトに変更を反映するには、
    [変更をコミットして Git にプッシュしてください
    ](https://docs.github.com/en/get-started/using-git/pushing-commits-to-a-remote-repository#about-git-push)。
    プッシュしても GitHub アプリによる変更のデプロイが確認できない場合は、
    [dashboard](https://dashboard.mintlify.com)から手動で
    ドキュメントを更新することもできます。
  </Accordion>
</AccordionGroup>

<div id="update-your-docs">
  ## ドキュメントを更新
</div>

MDX 構文と React コンポーネントを使って、ファイルに直接コンテンツを追加できます。用意されているコンポーネントはもちろん、独自のコンポーネントを作成することもできます。

<CardGroup>
  <Card title="ドキュメントのスタイルを設定" icon="paintbrush" href="/ja/settings/global">
    ブランドに合わせてカスタマイズし、ドキュメントの見栄えを向上させます。
  </Card>

  <Card title="API エンドポイントを追加" icon="square-code" href="/ja/api-playground/configuration">
    OpenAPI 仕様を実装し、API の利用者が操作できるようにします。
  </Card>

  <Card title="アナリティクスを統合" icon="chart-mixed" href="/ja/analytics/supported-integrations">
    ドキュメント上でのユーザー操作からインサイトを得られます。
  </Card>

  <Card title="カスタムドメインでホスト" icon="browser" href="/ja/settings/custom-domain/subdomain">
    自社サイトのサブドメインでドキュメントを公開できます。
  </Card>
</CardGroup>
